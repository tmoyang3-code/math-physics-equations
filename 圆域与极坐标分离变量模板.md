# 圆域与极坐标分离变量求解模板

## 一、从直角坐标到极坐标

### 1.1 坐标变换

$$x = r\cos\theta, \quad y = r\sin\theta$$

$$r = \sqrt{x^2 + y^2}, \quad \theta = \arctan\frac{y}{x}$$

### 1.2 拉普拉斯算子的变换

在直角坐标中：
$$\nabla^2 u = \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2}$$

在极坐标中：
$$\boxed{\nabla^2 u = \frac{\partial^2 u}{\partial r^2} + \frac{1}{r}\frac{\partial u}{\partial r} + \frac{1}{r^2}\frac{\partial^2 u}{\partial \theta^2}}$$

---

## 二、圆域问题的典型形式

### 2.1 圆内狄利克雷问题（最常见）

求解圆盘 $r < a$ 内的调和函数：

$$\begin{cases}
\nabla^2 u = \frac{\partial^2 u}{\partial r^2} + \frac{1}{r}\frac{\partial u}{\partial r} + \frac{1}{r^2}\frac{\partial^2 u}{\partial \theta^2} = 0, & r < a \\
u(a, \theta) = f(\theta), & 0 \leq \theta < 2\pi
\end{cases}$$

**边界条件的周期性**：
$$u(r, \theta) = u(r, \theta + 2\pi)$$
$$u(r, -\pi) = u(r, \pi)$$

### 2.2 圆内诺伊曼问题

$$\begin{cases}
\nabla^2 u = 0, & r < a \\
\frac{\partial u}{\partial r}\bigg|_{r=a} = g(\theta), & 0 \leq \theta < 2\pi
\end{cases}$$

### 2.3 圆环域问题

求解圆环 $b < r < a$ 内的问题：

$$\begin{cases}
\nabla^2 u = 0, & b < r < a \\
u(a, \theta) = f(\theta), & \text{外边界} \\
u(b, \theta) = g(\theta), & \text{内边界}
\end{cases}$$

---

## 三、与直角坐标分离变量的对比

| 特征 | 直角坐标 | 极坐标 |
|------|---------|--------|
| **坐标变量** | $x, y$（或 $x, t$） | $r, \theta$ |
| **分离形式** | $u(x,y) = X(x)Y(y)$ | $u(r,\theta) = R(r)\Theta(\theta)$ |
| **边界条件** | 两端固定 | 周期性：$\Theta(\theta) = \Theta(\theta + 2\pi)$ |
| **特征值问题** | $X'' + \lambda X = 0$ | $\Theta'' + n^2 \Theta = 0$ |
| **特征函数** | $\sin\frac{n\pi x}{L}$ | $\cos(n\theta), \sin(n\theta)$ |
| **径向方程** | 常系数 ODE | **欧拉方程**（变系数） |
| **径向解** | 指数函数/三角函数 | $r^n, r^{-n}$（或 $\ln r$） |

### 关键区别

1. **周期性边界条件**：极坐标的角向方程必须满足 $\Theta(\theta + 2\pi) = \Theta(\theta)$
2. **欧拉方程**：径向方程是 $r^2 R'' + rR' - n^2 R = 0$，不是常系数 ODE
3. **正则性条件**：圆心处 $r=0$ 不能有奇点，所以要舍去 $r^{-n}$ 和 $\ln r$ 项

---

## 四、解题步骤模板

### Step 1：分离变量

设 $u(r, \theta) = R(r)\Theta(\theta)$，代入拉普拉斯方程：

$$R''\Theta + \frac{1}{r}R'\Theta + \frac{1}{r^2}R\Theta'' = 0$$

两边乘以 $\frac{r^2}{R\Theta}$：

$$\frac{r^2 R'' + rR'}{R} = -\frac{\Theta''}{\Theta} = \lambda$$

得到两个常微分方程：

**角向方程**：
$$\Theta'' + \lambda \Theta = 0$$

**径向方程**：
$$r^2 R'' + rR' - \lambda R = 0 \quad \text{（欧拉方程）}$$

---

### Step 2：求解角向方程（特征值问题）

**边界条件**：周期性 $\Theta(\theta + 2\pi) = \Theta(\theta)$

#### 情况1：$\lambda = 0$

$$\Theta'' = 0 \implies \Theta = A + B\theta$$

周期性要求 $B = 0$，所以 $\Theta_0 = 1$（常数）

#### 情况2：$\lambda > 0$，设 $\lambda = n^2$

$$\Theta'' + n^2 \Theta = 0$$

$$\Theta = A\cos(n\theta) + B\sin(n\theta)$$

周期性要求 $n$ 为整数：$n = 1, 2, 3, \ldots$

#### 情况3：$\lambda < 0$（舍去）

$\lambda < 0$ 时解为指数函数，不满足周期性。

**特征值和特征函数**：
$$\lambda_n = n^2, \quad \Theta_n(\theta) = \begin{cases} 1, & n = 0 \\ \cos(n\theta), & n = 1, 2, 3, \ldots \\ \sin(n\theta), & n = 1, 2, 3, \ldots \end{cases}$$

---

### Step 3：求解径向方程（欧拉方程）

**方程**：$r^2 R'' + rR' - n^2 R = 0$

#### 情况1：$n = 0$

$$r^2 R'' + rR' = 0$$

令 $R = r^m$，得 $m(m-1) + m = 0 \implies m^2 = 0 \implies m = 0$

通解：$R_0 = C_0 + D_0 \ln r$

**正则性条件**：$r = 0$ 处有限，所以 $D_0 = 0$

$$\boxed{R_0 = 1}$$

#### 情况2：$n \geq 1$

令 $R = r^m$，代入得：
$$m(m-1) + m - n^2 = 0 \implies m^2 - n^2 = 0 \implies m = \pm n$$

通解：$R_n = C_n r^n + D_n r^{-n}$

**正则性条件**：$r = 0$ 处有限，所以 $D_n = 0$

$$\boxed{R_n = r^n}$$

#### 情况3：圆环域（$b < r < a$）

此时 $r = 0$ 不在区域内，两个解都要保留：

$$R_n = C_n r^n + D_n r^{-n}$$

---

### Step 4：写出一般解

#### 圆内问题（$r < a$）

$$u(r, \theta) = \frac{A_0}{2} + \sum_{n=1}^{\infty} r^n \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right]$$

#### 圆环问题（$b < r < a$）

$$u(r, \theta) = \frac{A_0}{2} + \sum_{n=1}^{\infty} \left[(C_n r^n + D_n r^{-n})\cos(n\theta) + (E_n r^n + F_n r^{-n})\sin(n\theta)\right]$$

---

### Step 5：应用边界条件确定系数

在 $r = a$ 处展开 $f(\theta)$：

$$f(\theta) = \frac{A_0}{2} + \sum_{n=1}^{\infty} a^n \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right]$$

利用傅里叶级数公式：

$$A_0 = \frac{1}{\pi} \int_0^{2\pi} f(\theta) \, d\theta$$

$$A_n = \frac{1}{\pi a^n} \int_0^{2\pi} f(\theta) \cos(n\theta) \, d\theta$$

$$B_n = \frac{1}{\pi a^n} \int_0^{2\pi} f(\theta) \sin(n\theta) \, d\theta$$

---

### Step 6：写出最终解

将系数代入一般解，得到最终答案。

---

## 五、泊松积分公式（快捷方法）

对于圆内狄利克雷问题，有**泊松积分公式**可以直接写出解：

$$u(r, \theta) = \frac{1}{2\pi} \int_0^{2\pi} \frac{a^2 - r^2}{a^2 - 2ar\cos(\theta - \varphi) + r^2} f(\varphi) \, d\varphi$$

**适用条件**：
- 圆盘 $r < a$ 内的调和函数
- 边界条件 $u(a, \theta) = f(\theta)$

**优点**：不需要计算傅里叶系数，直接积分得到解。

## 六、例题

### 例题 1：基本圆内狄利克雷问题

**题目**：
求解圆盘 $r < 1$ 内的拉普拉斯方程，边界条件：
$$u(1, \theta) = \cos(2\theta), \quad 0 \leq \theta < 2\pi$$

**解**：

#### Step 1-4：一般解形式

$$u(r, \theta) = \frac{A_0}{2} + \sum_{n=1}^{\infty} r^n \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right]$$

#### Step 5：应用边界条件

在 $r = 1$ 处：
$$u(1, \theta) = \frac{A_0}{2} + \sum_{n=1}^{\infty} \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right] = \cos(2\theta)$$

比较系数：
- $A_0 = 0$
- $A_2 = 1$，其他 $A_n = 0$
- 所有 $B_n = 0$

#### Step 6：最终解

$$\boxed{u(r, \theta) = r^2 \cos(2\theta)}$$

**验证**：
- 满足拉普拉斯方程 ✓
- $u(1, \theta) = \cos(2\theta)$ ✓

---

### 例题 2：边界条件为分段函数

**题目**：
求解圆盘 $r < 1$ 内的拉普拉斯方程，边界条件：
$$u(1, \theta) = \begin{cases} 1, & 0 \leq \theta < \pi \\ 0, & \pi \leq \theta < 2\pi \end{cases}$$

**解**：

#### Step 1-4：一般解形式

$$u(r, \theta) = \frac{A_0}{2} + \sum_{n=1}^{\infty} r^n \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right]$$

#### Step 5：计算傅里叶系数

$$A_0 = \frac{1}{\pi} \int_0^{2\pi} f(\theta) \, d\theta = \frac{1}{\pi} \int_0^{\pi} 1 \, d\theta = 1$$

$$A_n = \frac{1}{\pi} \int_0^{\pi} \cos(n\theta) \, d\theta = \frac{1}{\pi} \left[\frac{\sin(n\theta)}{n}\right]_0^{\pi} = 0$$

$$B_n = \frac{1}{\pi} \int_0^{\pi} \sin(n\theta) \, d\theta = \frac{1}{\pi} \left[-\frac{\cos(n\theta)}{n}\right]_0^{\pi} = \frac{1-(-1)^n}{n\pi}$$

$$B_n = \begin{cases} \frac{2}{n\pi}, & n \text{ 为奇数} \\ 0, & n \text{ 为偶数} \end{cases}$$

#### Step 6：最终解

$$\boxed{u(r, \theta) = \frac{1}{2} + \sum_{k=0}^{\infty} \frac{2r^{2k+1}}{(2k+1)\pi} \sin((2k+1)\theta)}$$

---

### 例题 3：半径为 $a$ 的圆盘

**题目**：
求解圆盘 $r < 2$ 内的拉普拉斯方程，边界条件：
$$u(2, \theta) = \theta, \quad 0 \leq \theta < 2\pi$$

**解**：

#### Step 1-4：一般解形式

$$u(r, \theta) = \frac{A_0}{2} + \sum_{n=1}^{\infty} \left(\frac{r}{2}\right)^n \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right]$$

**注意**：这里用 $\left(\frac{r}{2}\right)^n$ 而不是 $r^n$，因为 $a = 2$。

#### Step 5：计算傅里叶系数

$$A_0 = \frac{1}{\pi} \int_0^{2\pi} \theta \, d\theta = \frac{1}{\pi} \cdot 2\pi = 2$$

$$A_n = \frac{1}{\pi} \int_0^{2\pi} \theta \cos(n\theta) \, d\theta = 0$$

$$B_n = \frac{1}{\pi} \int_0^{2\pi} \theta \sin(n\theta) \, d\theta = \frac{2(-1)^{n+1}}{n}$$

#### Step 6：最终解

$$\boxed{u(r, \theta) = 1 + \sum_{n=1}^{\infty} \frac{2(-1)^{n+1}}{n} \left(\frac{r}{2}\right)^n \sin(n\theta)}$$

---

### 例题 4：圆环域问题

**题目**：
求解圆环 $1 < r < 2$ 内的拉普拉斯方程，边界条件：
$$u(1, \theta) = 0, \quad u(2, \theta) = \cos(\theta)$$

**解**：

#### Step 1-3：一般解形式（圆环）

由于 $r = 0$ 不在区域内，径向解保留两项：

$$u(r, \theta) = \frac{A_0}{2} + \sum_{n=1}^{\infty} \left[(C_n r^n + D_n r^{-n})\cos(n\theta) + (E_n r^n + F_n r^{-n})\sin(n\theta)\right]$$

#### Step 4：应用边界条件

**在 $r = 1$ 处**：$u(1, \theta) = 0$

$$\frac{A_0}{2} + \sum_{n=1}^{\infty} \left[(C_n + D_n)\cos(n\theta) + (E_n + F_n)\sin(n\theta)\right] = 0$$

所以：$A_0 = 0$，$C_n + D_n = 0$，$E_n + F_n = 0$

**在 $r = 2$ 处**：$u(2, \theta) = \cos(\theta)$

$$\sum_{n=1}^{\infty} \left[(C_n 2^n + D_n 2^{-n})\cos(n\theta) + (E_n 2^n + F_n 2^{-n})\sin(n\theta)\right] = \cos(\theta)$$

比较系数：
- $n = 1$：$C_1 \cdot 2 + D_1 \cdot \frac{1}{2} = 1$，$E_1 = F_1 = 0$
- $n \neq 1$：$C_n \cdot 2^n + D_n \cdot 2^{-n} = 0$

#### Step 5：求解系数

对于 $n = 1$：
$$\begin{cases} C_1 + D_1 = 0 \\ 2C_1 + \frac{1}{2}D_1 = 1 \end{cases}$$

解得：$C_1 = \frac{2}{3}$，$D_1 = -\frac{2}{3}$

#### Step 6：最终解

$$\boxed{u(r, \theta) = \left(\frac{2}{3}r - \frac{2}{3}r^{-1}\right)\cos(\theta) = \frac{2}{3}\left(r - \frac{1}{r}\right)\cos(\theta)}$$

---

### 例题 5：非齐次边界条件（傅里叶级数展开）

**题目**：
求解圆盘 $r < \pi$ 内的拉普拉斯方程，边界条件：
$$u(\pi, \theta) = |\theta|, \quad -\pi \leq \theta < \pi$$

**解**：

#### Step 1-4：一般解形式

$$u(r, \theta) = \frac{A_0}{2} + \sum_{n=1}^{\infty} \left(\frac{r}{\pi}\right)^n \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right]$$

#### Step 5：计算傅里叶系数

$|\theta|$ 是偶函数，所以 $B_n = 0$

$$A_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} |\theta| \, d\theta = \frac{2}{\pi} \int_0^{\pi} \theta \, d\theta = \pi$$

$$A_n = \frac{1}{\pi} \int_{-\pi}^{\pi} |\theta| \cos(n\theta) \, d\theta = \frac{2}{\pi} \int_0^{\pi} \theta \cos(n\theta) \, d\theta$$

分部积分：
$$A_n = \frac{2}{\pi} \left[\frac{\theta \sin(n\theta)}{n} + \frac{\cos(n\theta)}{n^2}\right]_0^{\pi} = \frac{2}{\pi n^2}(\cos(n\pi) - 1) = \frac{2((-1)^n - 1)}{\pi n^2}$$

$$A_n = \begin{cases} -\frac{4}{\pi n^2}, & n \text{ 为奇数} \\ 0, & n \text{ 为偶数} \end{cases}$$

#### Step 6：最终解

$$\boxed{u(r, \theta) = \frac{\pi}{2} - \frac{4}{\pi} \sum_{k=0}^{\infty} \frac{1}{(2k+1)^2} \left(\frac{r}{\pi}\right)^{2k+1} \cos((2k+1)\theta)}$$

---

### 例题 6：含时间的圆域热方程

**题目**：
求解圆盘 $r < 1$ 内的热传导方程：

$$\begin{cases}
\frac{\partial u}{\partial t} = \nabla^2 u, & r < 1, \ t > 0 \\
u(1, \theta) = \cos(\theta), & t > 0 \\
u(r, \theta, 0) = 0
\end{cases}$$

**解**：

#### Step 1：分离变量

设 $u(r, \theta, t) = R(r)\Theta(\theta)T(t)$

空间部分满足拉普拉斯方程，时间部分满足：

$$T' + \lambda T = 0$$

#### Step 2：空间问题

空间问题是圆内狄利克雷问题，特征值 $\lambda_n = n^2$

特征函数：
- 角向：$\cos(n\theta)$，$\sin(n\theta)$
- 径向：$r^n$

#### Step 3：时间解

$$T_n(t) = e^{-n^2 t}$$

#### Step 4：一般解

$$u(r, \theta, t) = \frac{A_0}{2} + \sum_{n=1}^{\infty} e^{-n^2 t} r^n \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right]$$

#### Step 5：应用边界条件和初始条件

**稳态解**（$t \to \infty$）：$u(1, \theta) = \cos(\theta)$

所以 $A_1 = 1$，其他系数为 0。

**初始条件**：$u(r, \theta, 0) = 0$

这与稳态解矛盾，说明需要求解瞬态问题。

**修正**：令 $u = v + w$，其中 $v$ 是稳态解，$w$ 满足齐次边界条件。

稳态解：$v(r, \theta) = r\cos(\theta)$

则 $w = u - v$ 满足：
$$w_t = \nabla^2 w, \quad w(1, \theta) = 0, \quad w(r, \theta, 0) = -r\cos(\theta)$$

#### Step 6：求解 $w$

$$w(r, \theta, t) = \sum_{n=1}^{\infty} e^{-n^2 t} r^n \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right]$$

初始条件：$w(r, \theta, 0) = -r\cos(\theta)$

所以 $A_1 = -1$，其他系数为 0。

$$w(r, \theta, t) = -e^{-t} r\cos(\theta)$$

#### Step 7：最终解

$$\boxed{u(r, \theta, t) = r\cos(\theta) - e^{-t} r\cos(\theta) = r(1 - e^{-t})\cos(\theta)}$$

---

### 例题 7：圆内诺伊曼问题

**题目**：
求解圆盘 $r < 1$ 内的拉普拉斯方程：

$$\begin{cases}
\nabla^2 u = 0, & r < 1 \\
\frac{\partial u}{\partial r}\bigg|_{r=1} = \cos(2\theta), & 0 \leq \theta < 2\pi
\end{cases}$$

**解**：

#### Step 1：一般解形式

$$u(r, \theta) = \frac{A_0}{2} + \sum_{n=1}^{\infty} r^n \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right]$$

#### Step 2：求导

$$\frac{\partial u}{\partial r} = \sum_{n=1}^{\infty} n r^{n-1} \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right]$$

#### Step 3：应用边界条件

在 $r = 1$ 处：
$$\sum_{n=1}^{\infty} n \left[A_n \cos(n\theta) + B_n \sin(n\theta)\right] = \cos(2\theta)$$

比较系数：
- $n = 2$：$2A_2 = 1 \implies A_2 = \frac{1}{2}$
- 其他 $A_n = 0$，所有 $B_n = 0$

**注意**：$A_0$ 无法由诺伊曼条件确定（相差一个常数）。

#### Step 4：最终解

$$\boxed{u(r, \theta) = \frac{A_0}{2} + \frac{r^2}{2}\cos(2\theta)}$$

其中 $A_0$ 是任意常数。

## 七、常见问题与技巧

### 7.1 如何选择坐标系？

| 问题类型 | 推荐坐标系 |
|---------|-----------|
| 矩形区域 | 直角坐标 $(x, y)$ |
| 圆形区域 | 极坐标 $(r, \theta)$ |
| 圆环区域 | 极坐标 $(r, \theta)$ |
| 球形区域 | 球坐标 $(r, \theta, \phi)$ |
| 扇形区域 | 极坐标 $(r, \theta)$ |

### 7.2 边界条件的处理

#### 圆内问题
- 只有一个边界：$r = a$
- 边界条件：$u(a, \theta) = f(\theta)$

#### 圆环问题
- 两个边界：$r = a$ 和 $r = b$
- 边界条件：$u(a, \theta) = f(\theta)$，$u(b, \theta) = g(\theta)$

#### 扇形问题
- 边界包括：$r = a$，$\theta = 0$，$\theta = \alpha$
- 角向方程需要满足齐次边界条件

### 7.3 傅里叶系数计算技巧

#### 偶函数
如果 $f(\theta)$ 是偶函数（$f(-\theta) = f(\theta)$）：
$$B_n = 0$$

#### 奇函数
如果 $f(\theta)$ 是奇函数（$f(-\theta) = -f(\theta)$）：
$$A_n = 0$$

#### 常见积分公式
$$\int_0^{2\pi} \cos(n\theta) \cos(m\theta) \, d\theta = \begin{cases} \pi, & n = m \\ 0, & n \neq m \end{cases}$$

$$\int_0^{2\pi} \sin(n\theta) \sin(m\theta) \, d\theta = \begin{cases} \pi, & n = m \\ 0, & n \neq m \end{cases}$$

$$\int_0^{2\pi} \cos(n\theta) \sin(m\theta) \, d\theta = 0$$

### 7.4 正则性条件

在圆心 $r = 0$ 处，解必须有界：
- 舍去 $r^{-n}$ 项（$n \geq 1$）
- 舍去 $\ln r$ 项

在圆环问题中（$r = 0$ 不在区域内），两个解都保留。

## 八、与直角坐标分离变量的类比

### 类比关系

| 直角坐标 | 极坐标 | 说明 |
|---------|--------|------|
| $x$ 方向 | $r$ 方向 | 径向 |
| $y$ 方向 | $\theta$ 方向 | 角向 |
| $X'' + \lambda X = 0$ | $\Theta'' + n^2 \Theta = 0$ | 特征值问题 |
| $\sin\frac{n\pi x}{L}$ | $\cos(n\theta), \sin(n\theta)$ | 特征函数 |
| 边界条件：两端固定 | 边界条件：周期性 | 决定特征值 |
| $Y'' - \lambda Y = 0$ | $r^2 R'' + rR' - n^2 R = 0$ | 径向方程 |
| 指数/三角函数 | $r^n$（幂函数） | 径向解 |

### 解题思路对比

**直角坐标**：
1. 分离变量 → 两个常系数 ODE
2. 求解特征值问题
3. 求解径向方程（常系数）
4. 叠加得一般解
5. 应用边界条件

**极坐标**：
1. 分离变量 → 角向（常系数）+ 径向（欧拉方程）
2. 求解角向特征值问题（周期性）
3. 求解径向欧拉方程
4. 叠加得一般解
5. 应用边界条件（傅里叶级数）

## 十、进阶内容

### 10.1 扇形区域

对于扇形区域 $0 < r < a$，$0 < \theta < \alpha$：

角向边界条件：$\Theta(0) = 0$，$\Theta(\alpha) = 0$

特征值：$\lambda_n = \left(\frac{n\pi}{\alpha}\right)^2$

特征函数：$\sin\left(\frac{n\pi\theta}{\alpha}\right)$

### 10.2 球坐标

对于球坐标 $(r, \theta, \phi)$，拉普拉斯方程的解涉及**勒让德多项式**：

$$u(r, \theta) = \sum_{n=0}^{\infty} r^n P_n(\cos\theta)$$

其中 $P_n$ 是 $n$ 次勒让德多项式。

### 10.3 贝塞尔函数

对于圆柱坐标 $(r, \theta, z)$，径向方程变为**贝塞尔方程**，解是贝塞尔函数 $J_n$ 和 $Y_n$。
