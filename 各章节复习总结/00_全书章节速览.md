# 数学物理方程 全书章节速览

> 用途：在十章详细总结之上提供一份顶层地图，方便快速回忆每章「研究什么 / 用什么方法 / 核心公式 / 与其他章的关系」。
> 来源：综合 `ppt/` 课件（00–09 章）与 `各章节复习总结/01–10` 的精华。

---

## 全书脉络

整本书围绕三类典型方程——**波动方程、热传导方程、Laplace 方程**——展开，按「建模 → 分类 → 求解 → 适定性 → 特殊函数」五条主线推进：

1. **建模**（第 0、1 章）：从物理规律导出方程，写出定解条件。
2. **分类**（第 2 章）：把任意二阶线性 PDE 化到三类标准型之一。
3. **求解**（第 3–7 章）：四类核心解法
   - 行波法（无界波动方程）
   - 分离变量法（有限区域）
   - 积分变换法（Fourier、Laplace）
   - Green 函数法（用基本解 + 边界）
4. **适定性**（第 8、9 章）：极值原理 + 能量积分法，证明唯一性与稳定性。
5. **特殊函数**（第 10 章）：在柱、球坐标下分离变量得到的 Bessel、Legendre 函数。

方法选择速查：

- 区域是 $\mathbb R^n$ 全空间 + 波动方程 → 行波法（d'Alembert / Kirchhoff / Poisson）
- 区域有限 + 齐次方程 + 齐次边界 → 分离变量法
- 半无界 / 无界 + 空间方向 → Fourier 变换
- $t\in[0,\infty)$ + 含初值 → Laplace 变换
- 一般边值问题 / 点源响应 → Green 函数法
- 圆 / 柱 / 球域 → Bessel / Legendre 函数
- 证明唯一性、稳定性 → 极值原理（椭圆、抛物）或能量法（双曲、抛物）

## 第 1 章 绪论与定解问题

**研究对象**：从物理规律导出三类典型方程，写出对应的定解问题。

**核心要点**

- 三类典型方程及其物理背景：
  - 波动 $u_{tt}=a^2\Delta u+f$（弦振动、声波）
  - 热传导 $u_t=a^2\Delta u+f$（扩散、传热）
  - Laplace / Poisson $\Delta u=0$ 或 $\Delta u=-F$（稳态场）
- 三类边界条件：Dirichlet（第一类）、Neumann（第二类）、Robin（第三类）
- 适定性 = 存在性 + 唯一性 + 稳定性
- 线性叠加原理：把含非齐次项 / 非齐次条件的复杂问题拆成几个简单问题

**考点**：根据物理情境（弦、杆、热传导棒、稳态膜）写出方程 + 定解条件；判断方程的线性性 / 齐次性 / 阶数。

---

## 第 2 章 二阶线性 PDE 的分类与标准型

**研究对象**：两个自变量的二阶线性 PDE

$$a_{11}u_{xx}+2a_{12}u_{xy}+a_{22}u_{yy}+\dots=f$$

**核心方法（化标准型五步）**

1. 算判别式 $\Delta=a_{12}^2-a_{11}a_{22}$。
2. 判类型：$\Delta>0$ 双曲、$\Delta=0$ 抛物、$\Delta<0$ 椭圆。
3. 解特征方程 $a_{11}(dy)^2-2a_{12}\,dx\,dy+a_{22}(dx)^2=0$ 得特征线 $\phi(x,y)=c$。
4. 作变量代换 $\xi=\phi_1,\eta=\phi_2$，把方程化到标准型。
5. 常系数情形再作指数变换 $u=Ve^{\lambda\xi+\mu\eta}$ 消去一阶项。

**与后续的连接**：第 3 章一维波动方程的 $\xi=x-at,\eta=x+at$ 就是这套方法的特例。

---

## 第 3 章 波动方程的初值问题与行波法

**研究对象**：无界 / 半无界区域上的波动方程 Cauchy 问题。

**三大公式**

- 一维 d'Alembert 公式：

$$u(x,t)=\tfrac12[\varphi(x-at)+\varphi(x+at)]+\tfrac1{2a}\int_{x-at}^{x+at}\psi(\xi)\,d\xi$$

- 三维 Kirchhoff 公式（球面平均法）。
- 二维 Poisson 公式（由三维降维法得到）。

**关键概念**：依赖区域、决定区域、影响区域、特征锥；有限传播速度 $a$；一维 / 三维「无后效」，二维「有后效」（Huygens 原理）。

**配套技巧**：半无界问题用奇 / 偶延拓；非齐次项用 Duhamel 齐次化原理。

---

## 第 4 章 分离变量法

**研究对象**：有限区域、齐次方程、齐次边界的初边值问题。**全书最核心的方法**，几乎每次考试都有大题。

**五步模板**

1. 设 $u(x,t)=X(x)T(t)$ 代入，得到两个 ODE。
2. 由齐次边界条件得到特征值问题 $X''+\lambda X=0$，求出 $\lambda_n$、$X_n(x)$。
3. 求 $T_n(t)$ 通解，得到一族特解 $u_n=X_nT_n$。
4. 叠加 $u(x,t)=\sum_n c_n X_n(x)T_n(t)$。
5. 用初始条件按特征函数正交性展开，定 $c_n$。

**扩展套路**

- 非齐次边界 → 先减去稳态解 / 线性函数化齐次。
- 非齐次方程 → 特征函数法（按 $X_n$ 展开 $f$ 与 $u$）。
- Sturm–Liouville 理论统一了「特征值 + 特征函数 + 正交性 + 展开定理」。

**与后续的连接**：第 5、6、7、10 章都依赖这一章的特征值思想。

---

## 第 5 章 Fourier 变换法

**研究对象**：空间变量在 $(-\infty,+\infty)$ 上的定解问题。

**核心思想**：用 Fourier 变换把对 $x$ 的求导变成乘 $i\alpha$，PDE 退化成对 $t$ 的 ODE，求解后再做逆变换。

**两条关键性质**

- 微分性质 $\mathcal F[f'(x)]=i\alpha\mathcal F[f]$。
- 卷积定理 $\mathcal F[f*g]=\mathcal F[f]\cdot\mathcal F[g]$。

**典型应用**：全直线热传导方程（Poisson 核）、全空间波动方程（与第 3 章公式呼应）。

**变体**：半直线 $[0,\infty)$ 用 Fourier 正弦 / 余弦变换，分别匹配 Dirichlet / Neumann 边界。

---

## 第 6 章 Laplace 变换法

**研究对象**：时间变量 $t\in[0,\infty)$ 上的定解问题，特别适合带初值的题目。

**核心思想**：$\bar f(s)=\int_0^\infty f(t)e^{-st}dt$；导数法则

$$\mathcal L[f'(t)]=s\bar f(s)-f(0)$$

天然吃掉初始条件，把对 $t$ 的 PDE 化为对 $s$ 的 ODE / 代数方程。

**关键工具**：常用变换表、平移定理、卷积定理、利用部分分式 + 查表做逆变换。

**与第 5 章对比**：Fourier 处理空间方向（无界）；Laplace 处理时间方向（半轴 + 初值）。两者经常组合使用。

---

## 第 7 章 Green 函数法

**研究对象**：一般区域上的边值问题，尤其是 Laplace / Poisson 方程的 Dirichlet 问题。

**核心三件套**

1. **Green 公式**：把区域内的微分方程转化为边界积分。
2. **基本解**：全空间中 $\Delta E=-\delta(x-y)$ 的解（三维 $E=\tfrac1{4\pi r}$，二维 $E=\tfrac1{2\pi}\ln\tfrac1r$）。
3. **Green 函数**：基本解 + 调和修正项，使其在边界为零，把 Dirichlet 问题写成

$$u(x)=-\int_{\partial\Omega}f(y)\frac{\partial G}{\partial n_y}\,dS_y+\int_\Omega F(y)G(x,y)\,dy$$

**构造手段**

- 镜像法：半空间、球内、上半平面、圆盘等对称区域。
- 特征函数展开：在矩形 / 长方体等可分离变量区域。

**δ 函数**：表示点源；基本解就是「点源响应」。

---

## 第 8 章 极值原理

**研究对象**：热方程、Laplace 方程解的「最大值 / 最小值在哪里取到」。

**两条核心定理**

- 抛物型极值原理：热方程解的最值在抛物边界（初始时刻 + 侧边界）上取到。
- 椭圆型极值原理：Laplace / Poisson 方程的调和函数最值在区域边界上取到。

**典型应用**

- 唯一性证明：两解之差满足齐次问题，由极值原理推出 $\equiv 0$。
- 稳定性 / 比较原理：估计 $|u|$ 上界，得 $|u|\le Me^{bt}$ 之类的不等式。
- 辅助函数法：构造 $w(x,t)$ 满足严格不等式，再反证。

**适用范围**：椭圆 + 抛物。**对双曲（波动）不适用**——需要第 9 章能量法补位。

---

## 第 9 章 能量积分法

**研究对象**：波动方程为主，兼顾热方程的适定性问题。

**能量法五步**

1. 构造能量泛函 $E(t)$（波方程含 $u_t^2,u_x^2$ 项；热方程含 $u^2$ 项）。
2. 对 $t$ 求导 $E'(t)$，把 $u_{tt}$ 或 $u_t$ 用方程换成空间导数。
3. 分部积分，把空间二阶导转出去，分成「边界项 + 体积项」。
4. 用边界条件让边界项 $\le 0$ 或 $=0$，得 $E'(t)\le 0$。
5. 由 $E(0)=0$（齐次初值）+ $E'\le 0$ → $E(t)\equiv 0$ → $u\equiv 0$。

**典型应用**

- 波动方程解的唯一性、对初值与右端的连续依赖性。
- 热方程的另一种唯一性证明（与第 8 章互为冗余但更通用）。

**与第 8 章对比**：极值原理是「逐点」论证，能量法是「积分」论证；后者对双曲方程依然有效。

---

## 第 10 章 Bessel 函数与 Legendre 函数

**研究对象**：在柱坐标 / 球坐标下分离变量得到的特殊函数。

**两套体系**

- 柱坐标分离变量 → Bessel 方程 → $J_\nu(x), Y_\nu(x)$ → Fourier–Bessel 级数。
- 球坐标（轴对称）分离变量 → Legendre 方程 → $P_n(x)$ → Fourier–Legendre 级数。

**关键性质**

- 正交性、模值公式、递推关系、母函数表达。
- $J_\nu$ 的零点决定圆 / 圆柱区域的特征值。
- $P_n$ 在 $[-1,1]$ 上正交，是球谐函数的轴对称部分。

**应用**：圆膜振动、圆柱内热传导、球内 Laplace 方程的 Dirichlet 问题。

**与第 4 章对比**：第 4 章用 $\{\sin\frac{n\pi x}{L}\}$ 这套三角特征函数系；第 10 章把场景换到圆 / 球，特征函数系换成 $J_n,P_n$，但「正交展开 + 系数公式」的结构完全平行——都是 Sturm–Liouville 理论的实例。
