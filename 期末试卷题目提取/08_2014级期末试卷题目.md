## 二、（12 分）方程化标准型（分参数讨论）

把方程

$$u_{xx} - 2\alpha u_{xy} - 3\alpha^2 u_{yy} + \alpha u_y + u_x = 0$$

化为标准型，其中 $\alpha$ 为常数（分别就 $\alpha = 0$ 和 $\alpha \neq 0$ 讨论）。

## 三、（12 分）波动方程 Dirichlet 边值定解问题

求下列定解问题的解：

$$\begin{cases} u_{tt} = a^2 u_{xx}, & 0 < x < \pi,\ t > 0, \\ u(x,0) = x,\quad u_t(x,0) = \cos 2x, & 0 \leq x \leq \pi, \\ u(0,t) = u(\pi,t) = 0, & t \geq 0. \end{cases}$$

## 四、（12 分）圆域 Dirichlet 问题

求解圆域上 Dirichlet（狄利克雷）问题的解：

$$\begin{cases} \Delta u = u_{rr} + r^{-1} u_r + r^{-2} u_{\theta\theta} = 0, & 0 < r < R,\ 0 \leq \theta \leq 2\pi, \\ u(R,\theta) = A\cos 2\theta + B\sin 3\theta, & 0 \leq \theta \leq 2\pi, \end{cases}$$

其中 $A, B$ 为常数。

## 五、（10 分）极值原理或能量积分法证唯一性

用极值原理或者能量积分方法证明下列初边值问题的古典（光滑）解若存在，则必唯一：

$$\begin{cases} u_t = c^2 u_{xx} + f(x,t), & 0 < x < L,\ t > 0, \\ u(x,0) = \varphi(x), & 0 \leq x \leq L, \\ -\alpha u_x(0,t) + \beta u(0,t) = g(t), & t \geq 0, \\ \alpha u_x(L,t) + \beta u(L,t) = h(t), & t \geq 0, \end{cases}$$

其中常数 $\alpha, \beta > 0$。

---

## 六、（12 分）Fourier 变换法求解热传导方程初值问题

用 Fourier 变换方法求解下列初值问题：

$$\begin{cases} u_t = c^2 u_{xx}, & -\infty < x < \infty,\ t > 0, \\ u(x,0) = x^2 + \sin 2x, & -\infty < x < \infty. \end{cases}$$

已知：若 $b > 0$，则

$$\int_0^\infty e^{-b x^2} \cos tx \, dx = \frac{\sqrt{\pi}}{2\sqrt{b}}\, e^{-\frac{t^2}{4b}}.$$

---

## 七、（12 分）Laplace 变换法求解 ODE

设 $a, b$ 为常数，函数 $f(t)$ 连续。利用 Laplace 变换方法求下列初值问题的解：

$$\begin{cases} y''(t) + \lambda y(t) = f(t), & t > 0, \\ y(0) = a,\quad y'(0) = b. \end{cases}$$

（提示：分别就 $\lambda = 0$、$\lambda > 0$、$\lambda < 0$ 三种情况讨论。）
