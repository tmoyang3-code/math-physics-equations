# 偏微分方程非齐次源项求解模板

## 一、问题形式

考虑定义在 $[0, L]$ 上的偏微分方程：

$$\frac{\partial^2 u}{\partial t^2} = a^2 \frac{\partial^2 u}{\partial x^2} + f(x,t) \quad \text{（非齐次波动方程）}$$

或

$$\frac{\partial u}{\partial t} = c^2 \frac{\partial^2 u}{\partial x^2} + f(x,t) \quad \text{（非齐次热方程）}$$

其中 $f(x,t)$ 是非齐次源项。

---

## 二、解题步骤模板

### Step 1：求解对应的齐次问题

**目的**：得到特征值 $\lambda_n$ 和特征函数 $X_n(x)$

将原方程的源项 $f(x,t)$ 置为 0，得到齐次问题：

$$\frac{\partial^2 u}{\partial t^2} = a^2 \frac{\partial^2 u}{\partial x^2}$$

边界条件：$u(0,t) = 0, \quad u(L,t) = 0$

**分离变量**：设 $u(x,t) = X(x)T(t)$，代入得：

$$\frac{T''}{a^2 T} = \frac{X''}{X} = -\lambda$$

**空间方程**：
$$X'' + \lambda X = 0, \quad X(0) = 0, \quad X(L) = 0$$

**特征值和特征函数**：
$$\lambda_n = \left(\frac{n\pi}{L}\right)^2, \quad X_n(x) = \sin\frac{n\pi x}{L}, \quad n = 1, 2, 3, \ldots$$

---

### Step 2：展开非齐次源项

**目的**：将 $f(x,t)$ 按特征函数展开

$$f(x,t) = \sum_{n=1}^{\infty} f_n(t) X_n(x)$$

**展开系数**：
$$f_n(t) = \frac{2}{L} \int_0^L f(x,t) \sin\frac{n\pi x}{L} \, dx$$

**注意**：
- $f_n(t)$ 是关于时间 $t$ 的函数
- 对每个 $n$ 都要计算积分

---

### Step 3：解时间方程

**目的**：求每个模态的时间函数 $T_n(t)$

将 $u(x,t) = \sum_{n=1}^{\infty} T_n(t) X_n(x)$ 代入原方程，得到：

#### 波动方程情况
$$T_n'' + \omega_n^2 T_n = f_n(t), \quad \omega_n = a\sqrt{\lambda_n} = \frac{an\pi}{L}$$

**解法**：特解 + 齐次通解
$$T_n(t) = \underbrace{A_n \cos(\omega_n t) + B_n \sin(\omega_n t)}_{\text{齐次通解}} + \underbrace{T_n^p(t)}_{\text{特解}}$$

**特解求法**（根据 $f_n(t)$ 的形式）：
- 若 $f_n(t) = C$（常数）：设 $T_n^p = D$，代入求 $D$
- 若 $f_n(t) = C\sin(\alpha t)$：设 $T_n^p = D\sin(\alpha t) + E\cos(\alpha t)$
- 若 $f_n(t) = Ce^{\beta t}$：设 $T_n^p = De^{\beta t}$

#### 热方程情况
$$T_n' + c^2\lambda_n T_n = f_n(t)$$

**解法**：用积分因子或常数变易法
$$T_n(t) = e^{-c^2\lambda_n t} \left[ \int_0^t f_n(\tau) e^{c^2\lambda_n \tau} \, d\tau + C_n \right]$$

---

### Step 4：应用初始条件

**目的**：确定常数 $A_n, B_n$（或 $C_n$）

将 $u(x,0) = \phi(x)$ 展开：
$$\phi(x) = \sum_{n=1}^{\infty} \phi_n \sin\frac{n\pi x}{L}$$

$$\phi_n = \frac{2}{L} \int_0^L \phi(x) \sin\frac{n\pi x}{L} \, dx$$

由 $T_n(0) = \phi_n$ 确定常数。

对于波动方程，还需利用 $u_t(x,0) = \psi(x)$ 确定另一个常数。

---

### Step 5：写出最终解

$$u(x,t) = \sum_{n=1}^{\infty} T_n(t) \sin\frac{n\pi x}{L}$$

---

## 三、例题

### 例题 1：非齐次热方程（常数源项）

**题目**：
$$\begin{cases}
\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2} + 1, & 0 < x < \pi, \ t > 0 \\
u(0,t) = u(\pi,t) = 0, & t > 0 \\
u(x,0) = 0, & 0 \leq x \leq \pi
\end{cases}$$

**解**：

#### Step 1：齐次问题的特征值和特征函数

空间方程：$X'' + \lambda X = 0, \quad X(0) = X(\pi) = 0$

特征值：$\lambda_n = n^2$，特征函数：$X_n(x) = \sin(nx)$

#### Step 2：展开源项 $f(x,t) = 1$

$$f_n(t) = \frac{2}{\pi} \int_0^{\pi} 1 \cdot \sin(nx) \, dx = \frac{2}{\pi} \left[-\frac{\cos(nx)}{n}\right]_0^{\pi} = \frac{2}{\pi n}(1 - (-1)^n)$$

$$f_n(t) = \begin{cases} \frac{4}{\pi n}, & n \text{ 为奇数} \\ 0, & n \text{ 为偶数} \end{cases}$$

#### Step 3：解时间方程

$$T_n' + n^2 T_n = f_n(t)$$

用积分因子 $e^{n^2 t}$：
$$T_n(t) = e^{-n^2 t} \int_0^t f_n(\tau) e^{n^2 \tau} \, d\tau + C_n e^{-n^2 t}$$

对于奇数 $n$，$f_n = \frac{4}{\pi n}$：
$$T_n(t) = \frac{4}{\pi n^3} (1 - e^{-n^2 t}) + C_n e^{-n^2 t}$$

#### Step 4：应用初始条件 $u(x,0) = 0$

$\phi_n = 0$，所以 $T_n(0) = 0$：
$$0 = 0 + C_n \implies C_n = 0$$

因此：
$$T_n(t) = \frac{4}{\pi n^3} (1 - e^{-n^2 t}), \quad n \text{ 为奇数}$$

#### Step 5：最终解

$$\boxed{u(x,t) = \frac{4}{\pi} \sum_{k=0}^{\infty} \frac{1 - e^{-(2k+1)^2 t}}{(2k+1)^3} \sin((2k+1)x)}$$

---

### 例题 2：非齐次波动方程（常数源项）

**题目**：
$$\begin{cases}
\frac{\partial^2 u}{\partial t^2} = \frac{\partial^2 u}{\partial x^2} + x, & 0 < x < \pi, \ t > 0 \\
u(0,t) = u(\pi,t) = 0, & t > 0 \\
u(x,0) = 0, \quad \frac{\partial u}{\partial t}(x,0) = 0, & 0 \leq x \leq \pi
\end{cases}$$

**解**：

#### Step 1：特征值和特征函数

$\lambda_n = n^2$，$X_n(x) = \sin(nx)$

#### Step 2：展开源项 $f(x,t) = x$

$$f_n(t) = \frac{2}{\pi} \int_0^{\pi} x \sin(nx) \, dx = \frac{2}{\pi} \left[-\frac{x\cos(nx)}{n} + \frac{\sin(nx)}{n^2}\right]_0^{\pi} = \frac{2}{n}(-1)^{n+1}$$

#### Step 3：解时间方程

$$T_n'' + n^2 T_n = \frac{2(-1)^{n+1}}{n}$$

**齐次通解**：$T_n^h = A_n \cos(nt) + B_n \sin(nt)$

**特解**：设 $T_n^p = D$（常数），代入：
$$n^2 D = \frac{2(-1)^{n+1}}{n} \implies D = \frac{2(-1)^{n+1}}{n^3}$$

**通解**：
$$T_n(t) = A_n \cos(nt) + B_n \sin(nt) + \frac{2(-1)^{n+1}}{n^3}$$

#### Step 4：应用初始条件

$u(x,0) = 0$：$\phi_n = 0$
$$T_n(0) = A_n + \frac{2(-1)^{n+1}}{n^3} = 0 \implies A_n = \frac{2(-1)^n}{n^3}$$

$u_t(x,0) = 0$：$\psi_n = 0$
$$T_n'(0) = n B_n = 0 \implies B_n = 0$$

因此：
$$T_n(t) = \frac{2(-1)^n}{n^3} \cos(nt) + \frac{2(-1)^{n+1}}{n^3} = \frac{2(-1)^{n+1}}{n^3} (1 - \cos(nt))$$

#### Step 5：最终解

$$\boxed{u(x,t) = \frac{2}{\pi} \sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n^3} (1 - \cos(nt)) \sin(nx)}$$

---

### 例题 3：非齐次热方程（含时间依赖源项）

**题目**：
$$\begin{cases}
\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2} + e^{-t}\sin(x), & 0 < x < \pi, \ t > 0 \\
u(0,t) = u(\pi,t) = 0, & t > 0 \\
u(x,0) = 0, & 0 \leq x \leq \pi
\end{cases}$$

**解**：

#### Step 1：特征值和特征函数

$\lambda_n = n^2$，$X_n(x) = \sin(nx)$

#### Step 2：展开源项 $f(x,t) = e^{-t}\sin(x)$

$$f_n(t) = \frac{2}{\pi} \int_0^{\pi} e^{-t}\sin(x) \sin(nx) \, dx = e^{-t} \cdot \frac{2}{\pi} \int_0^{\pi} \sin(x) \sin(nx) \, dx$$

由正交性：
$$f_n(t) = \begin{cases} e^{-t}, & n = 1 \\ 0, & n \neq 1 \end{cases}$$

#### Step 3：解时间方程（仅 $n=1$）

$$T_1' + T_1 = e^{-t}$$

积分因子 $e^t$：
$$\frac{d}{dt}(e^t T_1) = e^t \cdot e^{-t} = 1$$
$$e^t T_1 = t + C_1$$
$$T_1(t) = (t + C_1)e^{-t}$$

#### Step 4：应用初始条件 $u(x,0) = 0$

$T_1(0) = 0$：
$$0 = (0 + C_1) \cdot 1 \implies C_1 = 0$$

因此：$T_1(t) = t e^{-t}$

#### Step 5：最终解

$$\boxed{u(x,t) = t e^{-t} \sin(x)}$$

---

### 例题 4：非齐次波动方程（含时间依赖源项）

**题目**：
$$\begin{cases}
\frac{\partial^2 u}{\partial t^2} = \frac{\partial^2 u}{\partial x^2} + \sin(x)\cos(t), & 0 < x < \pi, \ t > 0 \\
u(0,t) = u(\pi,t) = 0, & t > 0 \\
u(x,0) = 0, \quad \frac{\partial u}{\partial t}(x,0) = 0, & 0 \leq x \leq \pi
\end{cases}$$

**解**：

#### Step 1：特征值和特征函数

$\lambda_n = n^2$，$X_n(x) = \sin(nx)$

#### Step 2：展开源项 $f(x,t) = \sin(x)\cos(t)$

$$f_n(t) = \begin{cases} \cos(t), & n = 1 \\ 0, & n \neq 1 \end{cases}$$

#### Step 3：解时间方程（仅 $n=1$）

$$T_1'' + T_1 = \cos(t)$$

**齐次通解**：$T_1^h = A_1 \cos(t) + B_1 \sin(t)$

**特解**：由于 $\cos(t)$ 是齐次解的形式，需要用**共振特解**：

设 $T_1^p = t(C\cos(t) + D\sin(t))$，代入：
$$T_1^{p''} + T_1^p = 2(-C\sin(t) + D\cos(t)) = \cos(t)$$

比较系数：$D = \frac{1}{2}$，$C = 0$

$$T_1^p = \frac{t}{2}\sin(t)$$

**通解**：
$$T_1(t) = A_1 \cos(t) + B_1 \sin(t) + \frac{t}{2}\sin(t)$$

#### Step 4：应用初始条件

$u(x,0) = 0$：$T_1(0) = A_1 = 0$

$u_t(x,0) = 0$：$T_1'(0) = B_1 + \frac{1}{2} = 0 \implies B_1 = -\frac{1}{2}$

$$T_1(t) = -\frac{1}{2}\sin(t) + \frac{t}{2}\sin(t) = \frac{t-1}{2}\sin(t)$$

#### Step 5：最终解

$$\boxed{u(x,t) = \frac{t-1}{2}\sin(t)\sin(x)}$$

---

### 例题 5：非齐次热方程（分段函数源项）

**题目**：
$$\begin{cases}
\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2} + x, & 0 < x < 1, \ t > 0 \\
u(0,t) = u(1,t) = 0, & t > 0 \\
u(x,0) = 0, & 0 \leq x \leq 1
\end{cases}$$

**解**：

#### Step 1：特征值和特征函数

$L = 1$，$\lambda_n = (n\pi)^2$，$X_n(x) = \sin(n\pi x)$

#### Step 2：展开源项 $f(x,t) = x$

$$f_n(t) = 2\int_0^1 x \sin(n\pi x) \, dx = 2\left[-\frac{x\cos(n\pi x)}{n\pi} + \frac{\sin(n\pi x)}{n^2\pi^2}\right]_0^1 = \frac{2(-1)^{n+1}}{n\pi}$$

#### Step 3：解时间方程

$$T_n' + n^2\pi^2 T_n = \frac{2(-1)^{n+1}}{n\pi}$$

特解（常数）：$T_n^p = \frac{2(-1)^{n+1}}{n^3\pi^3}$

通解：$T_n(t) = C_n e^{-n^2\pi^2 t} + \frac{2(-1)^{n+1}}{n^3\pi^3}$

#### Step 4：应用初始条件 $T_n(0) = 0$

$$C_n + \frac{2(-1)^{n+1}}{n^3\pi^3} = 0 \implies C_n = \frac{2(-1)^n}{n^3\pi^3}$$

$$T_n(t) = \frac{2(-1)^{n+1}}{n^3\pi^3}(1 - e^{-n^2\pi^2 t})$$

#### Step 5：最终解

$$\boxed{u(x,t) = \frac{2}{\pi^3} \sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n^3}(1 - e^{-n^2\pi^2 t})\sin(n\pi x)}$$

---

## 四、常见源项的展开系数

| 源项 $f(x,t)$ | 展开系数 $f_n(t)$ |
|--------------|------------------|
| 常数 $C$ | $C \cdot \frac{2(1-(-1)^n)}{n\pi}$（仅奇数项非零） |
| $x$ | $\frac{2(-1)^{n+1}}{n\pi}$ |
| $\sin(x)$ | $\delta_{n1}$（仅 $n=1$ 项） |
| $e^{-t}\sin(x)$ | $e^{-t}\delta_{n1}$ |
| $x^2$ | $\frac{2(-1)^n}{n\pi} + \frac{4(1-(-1)^n)}{n^3\pi^3}$ |

---

## 五、注意事项

1. **源项展开**：必须按照齐次问题的特征函数展开，不能用其他正交系
2. **特解形式**：根据 $f_n(t)$ 的具体形式选择合适的试探函数
3. **共振情况**：当源项频率与固有频率相同时，需要乘以 $t$ 的多项式
4. **初始条件**：始终用 $u(x,0) = \phi(x)$ 展开来确定常数
5. **收敛性**：最终级数解在 $t > 0$ 时收敛，$t = 0$ 时可能需要单独验证
