## `<a id="知识总结"></a>` 一、核心知识点总结

### 1. 傅里叶变换定义

**傅里叶变换**（像函数）：
$$F(\alpha) = \mathcal{F}[f(x)] = \int_{-\infty}^{+\infty} f(x) e^{-i\alpha x} dx$$

**傅里叶逆变换**（像原函数）：
$$f(x) = \mathcal{F}^{-1}[F(\alpha)] = \frac{1}{2\pi} \int_{-\infty}^{+\infty} F(\alpha) e^{i\alpha x} d\alpha$$

---

### 2. 傅里叶变换核心性质（必考！）

| 性质名称 | 公式 | 重要程度 | 应用场景 |
|:--------:|:----:|:--------:|:---------|
| **线性性质** | $\mathcal{F}[c_1 f + c_2 g] = c_1 F(\alpha) + c_2 G(\alpha)$ | ⭐⭐⭐ | 分解复杂函数 |
| **位移性质** | $\mathcal{F}[f(x-c)] = e^{-i\alpha c} F(\alpha)$ | ⭐⭐ | 函数平移 |
| **微分性质** | $\mathcal{F}[f^{(n)}(x)] = (i\alpha)^n F(\alpha)$ | ⭐⭐⭐⭐⭐ | **核心！求解微分方程** |
| **卷积定理** | $\mathcal{F}[(f*g)(x)] = F(\alpha) G(\alpha)$ | ⭐⭐⭐⭐⭐ | **核心！求逆变换** |

> **卷积定义**：$(f * g)(x) = \int_{-\infty}^{+\infty} f(x-\xi) g(\xi) d\xi$

---

### 3. 解热传导方程的基本思路

对热传导方程 $u_t = a^2u_{xx} + \dots$，对 $x$ 作傅里叶变换：
- $u_t \to \frac{dU}{dt}$（变换与时间无关）
- $u_x \to i\alpha U(\alpha, t)$
- $u_{xx} \to (i\alpha)^2 U = -\alpha^2 U$

变换后得到关于 $t$ 的 **一阶常微分方程**，解出 $U(\alpha, t)$ 后再求逆变换。

---

### 4. 热传导方程基本解（必须背！）

$$\mathcal{F}^{-1}\left[e^{-c^2 \alpha^2 t}\right] = \frac{1}{2c\sqrt{\pi t}} e^{-\frac{x^2}{4c^2 t}}$$

---

### 5. 卷积定理的应用

齐次热传导方程初值问题解的泊松公式：
$$u(x, t) = \frac{1}{2a\sqrt{\pi t}} \int_{-\infty}^{+\infty} \phi(\xi) \exp\left(-\frac{(x-\xi)^2}{4a^2 t}\right) d\xi$$

## `<a id="速查表"></a>` 二、特解速查表（考试必备！）

| 初值 $\varphi(x)$ | 热方程 $u_t = a^2u_{xx}$ 的解 | 验证 |
|:----------------:|:----------------------------:|:----:|
| 常数 $C$ | $C$（不变） | $u_t = 0, \, u_{xx} = 0$ ✓ |
| $x^2$ | $x^2 + 2a^2t$ | $u_t = 2a^2, \, u_{xx} = 2$ ✓ |
| $x^4$ | $x^4 + 12a^2 t x^2 + 12a^4 t^2$ | $u_t = 12a^2x^2 + 24a^4t, \, a^2u_{xx}$ 匹配 ✓ |
| $\cos\omega x$ | $e^{-a^2\omega^2 t}\cos\omega x$ | 指数衰减 + 频率不变 ✓ |
| $\sin\omega x$ | $e^{-a^2\omega^2 t}\sin\omega x$ | 指数衰减 + 频率不变 ✓ |

> 💡 **考试技巧**：遇到初值是多项式 + 三角函数的线性组合，直接用叠加原理分别查表，1 分钟出答案！

### 🔵 【基础型】D4. 2015 第六题（12 分）

**题目**：$u_t = c^2u_{xx}$，$u(x,0) = x^2 + \cos x$。

**📖 知识点定位**：第五章 §5.2–§5.3 Fourier 变换法解热方程

**解题思路**：
1. 初值是 $x^2 + \cos x$，拆为两部分
2. 查表直接写出解
3. 验证（可选，考试可省略）

**Step 1：查表得各部分解**

- **初值 $x^2$**：$v_1 = x^2 + 2c^2t$
- **初值 $\cos x$**：$v_2 = e^{-c^2t}\cos x$

**Step 2：叠加得最终解**

$$\boxed{u(x,t) = x^2 + 2c^2t + e^{-c^2t}\cos x}$$

---

### 🔵 【基础型】D5. 2014 第六题（12 分）

**题目**：$u_t = c^2u_{xx}$，$u(x,0) = x^2 + \sin 2x$。

**📖 知识点定位**：第五章 §5.2–§5.3 Fourier 变换法解热方程

**解题思路**：
- 拆为 $x^2$ 和 $\sin 2x$ 两部分
- 注意 $\sin 2x$ 中 $\omega = 2$，所以衰减系数是 $-c^2 \cdot 2^2 = -4c^2$

**Step 1：查表得各部分解**

- **初值 $x^2$**：$v_1 = x^2 + 2c^2t$
- **初值 $\sin 2x$**：$v_2 = e^{-4c^2t}\sin 2x$

**Step 2：叠加得最终解**

$$\boxed{u(x,t) = x^2 + 2c^2t + e^{-4c^2t}\sin 2x}$$

---

### 🔵 【基础型】D6. 2013 第六题（10 分）

**题目**：$u_t = c^2u_{xx}$，$u(x,0) = 1 + \cos x$。

**📖 知识点定位**：第五章 §5.2–§5.3 Fourier 变换法解热方程

**解题思路**：
- 拆为常数 1 和 $\cos x$ 两部分
- 常数是热方程的稳态解（导数为 0）

**Step 1：查表得各部分解**

- **初值 $1$**：$v_1 = 1$（常数不变）
- **初值 $\cos x$**：$v_2 = e^{-c^2t}\cos x$

**Step 2：叠加得最终解**

$$\boxed{u(x,t) = 1 + e^{-c^2t}\cos x}$$

> 💡 **母题发现**：2013-2015 三道题本质是同一道母题用不同初值！都是 $u_t = c^2u_{xx}$，只是初值不同。

---

### 🟡 【加源项】D1. 2020 第六题（12 分）

**题目**：$u_t = a^2u_{xx} + t^2$，$u(x,0) = x^2 + \sin x$。

**📖 知识点定位**：第五章 §5.3 Fourier 变换解定解问题

**解题思路**：
1. 源项 $t^2$ 与 $x$ 无关，利用叠加原理分离
2. 令 $u = v + w(t)$，$v$ 满足齐次方程，$w$ 只含 $t$
3. $w(t)$ 直接积分：$w' = t^2 \implies w = t^3/3$
4. $v$ 的初值不变，用特解凑法

**Step 1：叠加原理分离源项**

令 $u(x,t) = v(x,t) + w(t)$，则 $w_t = t^2 \implies w(t) = \frac{t^3}{3}$

$v$ 满足齐次热传导方程：
$$v_t = a^2v_{xx}, \quad v(x,0) = x^2 + \sin x$$

**Step 2：用特解凑法**

- 初值 $x^2$：$v_1 = x^2 + 2a^2t$
- 初值 $\sin x$：$v_2 = e^{-a^2t}\sin x$

**Step 3：合成最终解**

$$\boxed{u(x,t) = x^2 + 2a^2t + \frac{t^3}{3} + e^{-a^2t}\sin x}$$

> ✅ **验证**：$u_t = 2a^2 + t^2 - a^2 e^{-a^2t}\sin x$，$a^2u_{xx} = 2a^2 - a^2 e^{-a^2t}\sin x$，加 $t^2$ 匹配 ✓

---

### 🟡 【加反应项】D2. 2019 第六题（12 分）

**题目**：$u_t = a^2u_{xx} + cu$，$u(x,0) = x^2 + \cos x$。

**📖 知识点定位**：第五章 §5.3 Fourier 变换解热传导方程

**解题思路**：
1. 反应项 $cu$ 可通过指数变换消去
2. 令 $v = e^{-ct}u$，则 $u = e^{ct}v$
3. 代入得 $v_t = a^2v_{xx}$（齐次！）
4. 解 $v$ 后回代

**Step 1：指数变换消去反应项**

令 $v(x,t) = e^{-ct}u(x,t)$，则：
$$v_t = e^{-ct}(u_t - cu) = e^{-ct}(a^2u_{xx}) = a^2v_{xx}$$

初始条件：$v(x,0) = x^2 + \cos x$

**Step 2：解齐次方程**

- 初值 $x^2$：$v_1 = x^2 + 2a^2t$
- 初值 $\cos x$：$v_2 = e^{-a^2t}\cos x$

**Step 3：回代 $u = e^{ct}v$**

$$\boxed{u(x,t) = e^{ct}(x^2 + 2a^2t + e^{-a^2t}\cos x)}$$

> ✅ **验证**：$u_t = cu + e^{ct}(2a^2 - a^2 e^{-a^2t}\cos x)$，$a^2u_{xx} = a^2 e^{ct}(2 - e^{-a^2t}\cos x)$，匹配 ✓

---

### 🟠 【加对流-反应项】D3. 2018 第七题（5 分）

**题目**：$u_t = a^2u_{xx} + bu_x + cu$，$u(x,0) = \varphi(x)$。求一般解和 $\varphi(x)=\sin x$ 的特解。

**📖 知识点定位**：第五章 §5.3 Fourier 变换法

**解题思路**：
1. 直接对 $x$ 作 Fourier 变换
2. $u_{xx} \to -\alpha^2 U$，$u_x \to i\alpha U$
3. 解 ODE 得 $U(\alpha, t) = \Phi(\alpha) e^{(-a^2\alpha^2 + ib\alpha + c)t}$
4. 逆变换：$e^{ib\alpha t}$ 是平移 $x \to x+bt$，$e^{ct}$ 是指数因子，$e^{-a^2\alpha^2t}$ 是热核

**Step 1：对 $x$ 做 Fourier 变换**

$$\hat u_t = (-a^2\alpha^2 + ib\alpha + c)\hat u$$

解得：
$$\hat u(\alpha,t) = \hat\varphi(\alpha)\,e^{(-a^2\alpha^2+ib\alpha+c)t}$$

**Step 2：求逆变换得一般解**

指数可拆为：$e^{ct} \cdot e^{ib\alpha t} \cdot e^{-a^2\alpha^2 t}$

- $e^{ib\alpha t}$ 的逆变换是平移 $x \to x+bt$
- $e^{-a^2\alpha^2 t}$ 的逆变换是热核

因此：

$$\boxed{u(x,t) = \frac{e^{ct}}{2a\sqrt{\pi t}}\int_{-\infty}^{\infty}\varphi(\xi)\exp\!\left(-\frac{(x+bt-\xi)^2}{4a^2t}\right)d\xi}$$

**Step 3：当 $\varphi(x)=\sin x$ 时求特解**

$\sin x$ 的 Fourier 变换是两个 $\delta$ 函数之和，筛选后得：

$$\boxed{u(x,t) = e^{(c-a^2)t}\sin(x+bt)}$$

> 🏆 **物理含义**：
> - $e^{-a^2t}$ = 扩散衰减
> - $\sin(x+bt)$ = 对流平移（以速度 $-b$ 移动）
> - $e^{ct}$ = 反应增长/衰减

### 傅里叶变换热传导方程变体谱系图

```
基础型：u_t = a²u_xx
│
├── 初值变体：φ(x) 变化
│   ├── 常数 + 三角函数：D6 (1 + cosx)
│   ├── 二次函数 + 正弦：D5 (x² + sin2x)

│   └── D1：f(t) = t²，与 x 无关
│
├── 加反应项：u_t = a²u_xx + cu
│   └── D2：指数变换 v = e^(-ct)u
│
└── 加对流-反应项：u_t = a²u_xx + bu_x + cu
    └── D3：Fourier 变换直接解，卷积定积分
```

## `<a id="模板"></a>` 五、解题通用模板

### 模板 A：齐次型（仅初值变化）

**题目形式**：$u_t = c^2u_{xx}$，$u(x,0) = \varphi(x)$

**解题步骤**：

```
第1步：分解初值
   φ(x) = 多项式项 + 三角函数项
   
第2步：查表写出各分量解
   - 常数 C → C
   - x² → x² + 2c²t
   - cosωx → e^(-c²ω²t)cosωx
   - sinωx → e^(-c²ω²t)sinωx
   
第3步：叠加
   u = Σ各分量解
   
第4步：验证（可选）
   求 u_t 和 c²u_xx 是否相等
```

---

### 模板 B：加源项型

**题目形式**：$u_t = a^2u_{xx} + f(t)$

**解题步骤**：

```
第1步：分离变量
   令 u = v(x,t) + w(t)
   
第2步：求 w(t)
   w' = f(t) → 积分求 w(t)
   
第3步：v 满足齐次方程，按模板A解
   
第4步：合成 u = v + w
```

---

### 模板 C：加反应项型

**题目形式**：$u_t = a^2u_{xx} + cu$

**解题步骤**：

```
第1步：指数变换消去 cu
   令 v = e^(-ct)u，则 v_t = a²v_xx
   
第2步：按模板A解 v
   
第3步：回代 u = e^(ct)v
```

---

### 模板 D：加对流-反应项（一般解）

**题目形式**：$u_t = a^2u_{xx} + bu_x + cu$

**解题步骤**：

```
第1步：对 x 作 Fourier 变换
   U_t = (-a²α² + ibα + c)U
   
第2步：解 ODE
   U(α,t) = Φ(α)e^(-a²α²+ibα+c)t
   
第3步：逆变换（卷积定理）
   - e^(ibαt) → 平移 x → x+bt
   - e^(-a²α²t) → 热核
   - e^(ct) → 指数因子
   
第4步：写出卷积积分
   u = 热核 * φ(x+bt)
```

## ⚠️ 常见易错点提醒

1. **三角函数的频率别搞错**！$\sin 2x$ 的衰减是 $e^{-4c^2t}$，不是 $e^{-c^2t}$

2. **加反应项最后别忘了乘 $e^{ct}$**！很多同学解完 $v$ 就忘了回代

3. **对流项是平移 $x \to x+bt$**！注意符号：$bu_x$ 对应正号平移

4. **初始条件别带错**！分离变量后 $v$ 的初始条件和 $u$ 相同
