# 非齐次边界条件求解模板

## 一、问题形式

考虑定义在 $[0, L]$ 上的偏微分方程，带有**非齐次边界条件**：

$$\begin{cases}
\frac{\partial^2 u}{\partial t^2} = a^2 \frac{\partial^2 u}{\partial x^2}, & 0 < x < L, \ t > 0 \\
u(0,t) = \alpha(t), \quad u(L,t) = \beta(t), & t > 0 \\
u(x,0) = \phi(x), \quad \frac{\partial u}{\partial t}(x,0) = \psi(x)
\end{cases}$$

或热方程：

$$\begin{cases}
\frac{\partial u}{\partial t} = c^2 \frac{\partial^2 u}{\partial x^2}, & 0 < x < L, \ t > 0 \\
u(0,t) = \alpha(t), \quad u(L,t) = \beta(t), & t > 0 \\
u(x,0) = \phi(x)
\end{cases}$$

**关键**：边界条件 $\alpha(t)$ 和 $\beta(t)$ 不为零，无法直接分离变量。

---

## 二、边界齐次化原理

### 核心思想

引入**辅助函数** $v(x,t)$，构造新函数：

$$w(x,t) = u(x,t) - v(x,t)$$

使得 $w(x,t)$ 满足**齐次边界条件**：

$$w(0,t) = 0, \quad w(L,t) = 0$$

### 辅助函数的要求

1. **满足边界条件**：
   - $v(0,t) = \alpha(t)$
   - $v(L,t) = \beta(t)$

2. **形式简单**：通常选为 $x$ 的低次多项式（一次或二次）

3. **时间依赖性**：可以依赖于 $t$，方便处理

---

## 三、解题步骤模板

### Step 1：构造辅助函数 $v(x,t)$

**目标**：找到满足边界条件的简单函数

#### 情况1：两个边界都非齐次

$$v(x,t) = \alpha(t) + \frac{x}{L}[\beta(t) - \alpha(t)]$$

**验证**：
- $v(0,t) = \alpha(t) + 0 = \alpha(t)$ ✓
- $v(L,t) = \alpha(t) + \beta(t) - \alpha(t) = \beta(t)$ ✓

#### 情况2：仅左边界非齐次（$\beta(t) = 0$）

$$v(x,t) = \alpha(t)\left(1 - \frac{x}{L}\right)$$

#### 情况3：仅右边界非齐次（$\alpha(t) = 0$）

$$v(x,t) = \beta(t)\frac{x}{L}$$

#### 情况4：需要更简单的形式

如果边界条件复杂，可以选择满足边界条件的**任意简单函数**，不一定是最简线性函数。

---

### Step 2：变换原问题

令 $w(x,t) = u(x,t) - v(x,t)$，则 $u(x,t) = w(x,t) + v(x,t)$

**变换后的方程**：

将 $u = w + v$ 代入原方程，得到 $w$ 满足的方程。

#### 波动方程情况

原方程：$u_{tt} = a^2 u_{xx}$

$$w_{tt} + v_{tt} = a^2(w_{xx} + v_{xx})$$

$$w_{tt} = a^2 w_{xx} + (a^2 v_{xx} - v_{tt})$$

令 $F(x,t) = a^2 v_{xx} - v_{tt}$，则：

$$\boxed{w_{tt} = a^2 w_{xx} + F(x,t)}$$

#### 热方程情况

原方程：$u_t = c^2 u_{xx}$

$$w_t + v_t = c^2(w_{xx} + v_{xx})$$

$$w_t = c^2 w_{xx} + (c^2 v_{xx} - v_t)$$

令 $F(x,t) = c^2 v_{xx} - v_t$，则：

$$\boxed{w_t = c^2 w_{xx} + F(x,t)}$$

---

### Step 3：确定 $w$ 的初始条件

#### 波动方程情况

$$w(x,0) = u(x,0) - v(x,0) = \phi(x) - v(x,0)$$

$$w_t(x,0) = u_t(x,0) - v_t(x,0) = \psi(x) - v_t(x,0)$$

#### 热方程情况

$$w(x,0) = u(x,0) - v(x,0) = \phi(x) - v(x,0)$$

---

### Step 4：求解 $w$ 的问题

现在 $w(x,t)$ 满足：
- **齐次边界条件**：$w(0,t) = w(L,t) = 0$
- **可能非齐次方程**：$w_{tt} = a^2 w_{xx} + F(x,t)$ 或 $w_t = c^2 w_{xx} + F(x,t)$

使用**分离变量法**或**特征函数法**求解。

---

### Step 5：写出最终解

$$u(x,t) = w(x,t) + v(x,t)$$

---

## 四、辅助函数构造技巧

### 常用辅助函数形式

| 边界条件 | 辅助函数 $v(x,t)$ |
|---------|------------------|
| $u(0,t) = \alpha(t), \ u(L,t) = 0$ | $\alpha(t)(1 - \frac{x}{L})$ |
| $u(0,t) = 0, \ u(L,t) = \beta(t)$ | $\beta(t)\frac{x}{L}$ |
| $u(0,t) = \alpha(t), \ u(L,t) = \beta(t)$ | $\alpha(t) + \frac{x}{L}[\beta(t) - \alpha(t)]$ |
| $u_x(0,t) = \alpha(t), \ u(L,t) = 0$ | $\alpha(t)(x - \frac{x^2}{2L})$ |
| $u(0,t) = \alpha(t), \ u_x(L,t) = 0$ | $\alpha(t)(1 - \frac{x}{2L})$ |

### 选择辅助函数的原则

1. **优先选线性函数**：$v(x,t) = A(t) + B(t)x$
2. **二次函数**：当边界涉及导数时使用
3. **尽量简单**：越简单，变换后的源项 $F(x,t)$ 越容易处理
4. **可分离变量**：如果 $v(x,t)$ 能写成 $X(x)T(t)$ 的形式，计算更方便

---

## 五、例题

### 例题 1：波动方程 - 左边界非齐次

**题目**：
$$\begin{cases}
\frac{\partial^2 u}{\partial t^2} = \frac{\partial^2 u}{\partial x^2}, & 0 < x < \pi, \ t > 0 \\
u(0,t) = \sin(t), \quad u(\pi,t) = 0, & t > 0 \\
u(x,0) = 0, \quad \frac{\partial u}{\partial t}(x,0) = 0
\end{cases}$$

**解**：

#### Step 1：构造辅助函数

$$v(x,t) = \sin(t)\left(1 - \frac{x}{\pi}\right)$$

验证：$v(0,t) = \sin(t)$，$v(\pi,t) = 0$ ✓

#### Step 2：变换原问题

$$w(x,t) = u(x,t) - v(x,t)$$

计算 $v$ 的偏导数：
- $v_{tt} = -\sin(t)(1 - \frac{x}{\pi})$
- $v_{xx} = 0$

源项：
$$F(x,t) = a^2 v_{xx} - v_{tt} = 0 - [-\sin(t)(1 - \frac{x}{\pi})] = \sin(t)(1 - \frac{x}{\pi})$$

变换后的问题：
$$w_{tt} = w_{xx} + \sin(t)(1 - \frac{x}{\pi})$$

边界条件：$w(0,t) = 0$，$w(\pi,t) = 0$ ✓

#### Step 3：初始条件

$$w(x,0) = u(x,0) - v(x,0) = 0 - 0 = 0$$

$$w_t(x,0) = u_t(x,0) - v_t(x,0) = 0 - \cos(t)(1 - \frac{x}{\pi})\big|_{t=0} = -(1 - \frac{x}{\pi})$$

#### Step 4：求解 $w$ 的问题

特征值：$\lambda_n = n^2$，特征函数：$\sin(nx)$

展开源项：
$$F(x,t) = \sin(t)(1 - \frac{x}{\pi}) = \sin(t) \sum_{n=1}^{\infty} f_n \sin(nx)$$

$$f_n = \frac{2}{\pi} \int_0^{\pi} (1 - \frac{x}{\pi}) \sin(nx) \, dx = \frac{2}{n\pi}$$

展开初始条件：
$$w(x,0) = 0 \implies A_n = 0$$

$$w_t(x,0) = -(1 - \frac{x}{\pi}) = \sum_{n=1}^{\infty} B_n n \sin(nx)$$

$$B_n = \frac{2}{n\pi} \cdot \frac{2}{\pi} \int_0^{\pi} (1 - \frac{x}{\pi}) \sin(nx) \, dx = \frac{4}{n^2\pi^2}$$

**时间方程**：
$$T_n'' + n^2 T_n = \frac{2}{n\pi}\sin(t)$$

特解：设 $T_n^p = C\sin(t)$（$n \neq 1$ 时）
$$(n^2 - 1)C = \frac{2}{n\pi} \implies C = \frac{2}{n\pi(n^2-1)}$$

通解：$T_n(t) = A_n\cos(nt) + B_n\sin(nt) + \frac{2}{n\pi(n^2-1)}\sin(t)$

应用初始条件 $T_n(0) = 0$：$A_n = 0$

应用初始条件 $T_n'(0) = B_n n$：
$$B_n n + \frac{2}{n\pi(n^2-1)} = \frac{4}{n^2\pi^2}$$

$$B_n = \frac{4}{n^3\pi^2} - \frac{2}{n^2\pi(n^2-1)}$$

最终 $w(x,t)$ 的表达式（略）

#### Step 5：最终解

$$u(x,t) = w(x,t) + \sin(t)(1 - \frac{x}{\pi})$$

### 例题 2：热方程 - 两个边界都非齐次

**题目**：
$$\begin{cases}
\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2}, & 0 < x < \pi, \ t > 0 \\
u(0,t) = e^{-t}, \quad u(\pi,t) = e^{-t}, & t > 0 \\
u(x,0) = \sin(x)
\end{cases}$$

**解**：

#### Step 1：构造辅助函数

两个边界都是 $e^{-t}$，选择常数函数：
$$v(x,t) = e^{-t}$$

验证：$v(0,t) = e^{-t}$，$v(\pi,t) = e^{-t}$ ✓

#### Step 2：变换原问题

$$w(x,t) = u(x,t) - e^{-t}$$

计算 $v$ 的偏导数：
- $v_t = -e^{-t}$
- $v_{xx} = 0$

源项：
$$F(x,t) = c^2 v_{xx} - v_t = 0 - (-e^{-t}) = e^{-t}$$

变换后的问题：
$$w_t = w_{xx} + e^{-t}$$

边界条件：$w(0,t) = 0$，$w(\pi,t) = 0$ ✓

#### Step 3：初始条件

$$w(x,0) = u(x,0) - v(x,0) = \sin(x) - 1$$

#### Step 4：求解 $w$ 的问题

特征值：$\lambda_n = n^2$，特征函数：$\sin(nx)$

展开初始条件：
$$w(x,0) = \sin(x) - 1 = \sum_{n=1}^{\infty} \phi_n \sin(nx)$$

$$\phi_1 = 1, \quad \phi_n = \frac{2}{\pi} \int_0^{\pi} (-1) \sin(nx) \, dx = \frac{2(1-(-1)^n)}{n\pi} \ (n \geq 2)$$

展开源项：
$$e^{-t} = \sum_{n=1}^{\infty} f_n(t) \sin(nx)$$

$$f_n(t) = \frac{2(1-(-1)^n)}{n\pi} e^{-t}$$

**时间方程**：
$$T_n' + n^2 T_n = f_n(t) = \frac{2(1-(-1)^n)}{n\pi} e^{-t}$$

用积分因子 $e^{n^2 t}$：
$$T_n(t) = e^{-n^2 t} \left[ \int_0^t \frac{2(1-(-1)^n)}{n\pi} e^{-\tau} e^{n^2 \tau} \, d\tau + C_n \right]$$

$$= e^{-n^2 t} \left[ \frac{2(1-(-1)^n)}{n\pi(n^2-1)} (e^{(n^2-1)t} - 1) + C_n \right]$$

应用初始条件 $T_n(0) = \phi_n$：
- 对于 $n=1$：$T_1(0) = 1$，直接求解得 $T_1(t) = e^{-t}$
- 对于 $n \geq 2$：确定 $C_n$

最终得到 $w(x,t)$ 的级数解。

#### Step 5：最终解

$$u(x,t) = w(x,t) + e^{-t}$$

---

### 例题 3：波动方程 - 右边界非齐次

**题目**：
$$\begin{cases}
\frac{\partial^2 u}{\partial t^2} = 4\frac{\partial^2 u}{\partial x^2}, & 0 < x < 1, \ t > 0 \\
u(0,t) = 0, \quad u(1,t) = t, & t > 0 \\
u(x,0) = 0, \quad \frac{\partial u}{\partial t}(x,0) = 0
\end{cases}$$

**解**：

#### Step 1：构造辅助函数

$$v(x,t) = t \cdot x$$

验证：$v(0,t) = 0$，$v(1,t) = t$ ✓

#### Step 2：变换原问题

$$w(x,t) = u(x,t) - tx$$

计算 $v$ 的偏导数：
- $v_{tt} = 0$
- $v_{xx} = 0$

源项：
$$F(x,t) = a^2 v_{xx} - v_{tt} = 0$$

变换后的问题：
$$w_{tt} = 4w_{xx}$$

边界条件：$w(0,t) = 0$，$w(1,t) = 0$ ✓

#### Step 3：初始条件

$$w(x,0) = u(x,0) - v(x,0) = 0 - 0 = 0$$

$$w_t(x,0) = u_t(x,0) - v_t(x,0) = 0 - x = -x$$

#### Step 4：求解 $w$ 的问题

特征值：$\lambda_n = (n\pi)^2$，特征函数：$\sin(n\pi x)$

展开初始条件：
$$w_t(x,0) = -x = \sum_{n=1}^{\infty} B_n n\pi \sin(n\pi x)$$

$$B_n = \frac{2}{n\pi} \int_0^1 (-x) \sin(n\pi x) \, dx = \frac{2(-1)^n}{n^2\pi^2}$$

**通解**（$w_{tt} = 4w_{xx}$，$a=2$）：
$$w(x,t) = \sum_{n=1}^{\infty} B_n \sin(n\pi x) \sin(2n\pi t)$$

$$= \sum_{n=1}^{\infty} \frac{2(-1)^n}{n^2\pi^2} \sin(n\pi x) \sin(2n\pi t)$$

#### Step 5：最终解

$$\boxed{u(x,t) = tx + \sum_{n=1}^{\infty} \frac{2(-1)^n}{n^2\pi^2} \sin(n\pi x) \sin(2n\pi t)}$$

---

### 例题 4：热方程 - 左边界非齐次，含时间依赖

**题目**：
$$\begin{cases}
\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2}, & 0 < x < 1, \ t > 0 \\
u(0,t) = t^2, \quad u(1,t) = 0, & t > 0 \\
u(x,0) = 0
\end{cases}$$

**解**：

#### Step 1：构造辅助函数

$$v(x,t) = t^2(1-x)$$

验证：$v(0,t) = t^2$，$v(1,t) = 0$ ✓

#### Step 2：变换原问题

$$w(x,t) = u(x,t) - t^2(1-x)$$

计算 $v$ 的偏导数：
- $v_t = 2t(1-x)$
- $v_{xx} = 0$

源项：
$$F(x,t) = c^2 v_{xx} - v_t = 0 - 2t(1-x) = -2t(1-x)$$

变换后的问题：
$$w_t = w_{xx} - 2t(1-x)$$

边界条件：$w(0,t) = 0$，$w(1,t) = 0$ ✓

#### Step 3：初始条件

$$w(x,0) = u(x,0) - v(x,0) = 0 - 0 = 0$$

#### Step 4：求解 $w$ 的问题

特征值：$\lambda_n = (n\pi)^2$，特征函数：$\sin(n\pi x)$

展开源项：
$$-2t(1-x) = \sum_{n=1}^{\infty} f_n(t) \sin(n\pi x)$$

$$f_n(t) = -2t \cdot \frac{2}{n\pi} = -\frac{4t}{n\pi}$$

**时间方程**：
$$T_n' + n^2\pi^2 T_n = -\frac{4t}{n\pi}$$

用积分因子 $e^{n^2\pi^2 t}$：
$$T_n(t) = e^{-n^2\pi^2 t} \left[ \int_0^t -\frac{4\tau}{n\pi} e^{n^2\pi^2 \tau} \, d\tau + C_n \right]$$

分部积分：
$$\int_0^t \tau e^{n^2\pi^2 \tau} \, d\tau = \frac{1}{n^2\pi^2}\left[\tau e^{n^2\pi^2 \tau} - \frac{e^{n^2\pi^2 \tau}}{n^2\pi^2}\right]_0^t$$

$$= \frac{1}{n^2\pi^2}\left[t e^{n^2\pi^2 t} - \frac{e^{n^2\pi^2 t} - 1}{n^2\pi^2}\right]$$

应用初始条件 $T_n(0) = 0$：$C_n = 0$

$$T_n(t) = -\frac{4}{n\pi \cdot n^2\pi^2}\left[t - \frac{1 - e^{-n^2\pi^2 t}}{n^2\pi^2}\right]$$

$$= -\frac{4}{n^3\pi^3}\left[t - \frac{1 - e^{-n^2\pi^2 t}}{n^2\pi^2}\right]$$

#### Step 5：最终解

$$\boxed{u(x,t) = t^2(1-x) - \frac{4}{\pi^3} \sum_{n=1}^{\infty} \frac{1}{n^3} \left[t - \frac{1 - e^{-n^2\pi^2 t}}{n^2\pi^2}\right] \sin(n\pi x)}$$

### 例题 5：波动方程 - 两个边界都非齐次

**题目**：
$$\begin{cases}
\frac{\partial^2 u}{\partial t^2} = \frac{\partial^2 u}{\partial x^2}, & 0 < x < \pi, \ t > 0 \\
u(0,t) = \cos(t), \quad u(\pi,t) = \sin(t), & t > 0 \\
u(x,0) = \frac{x}{\pi}, \quad \frac{\partial u}{\partial t}(x,0) = 0
\end{cases}$$

**解**：

#### Step 1：构造辅助函数

$$v(x,t) = \cos(t) + \frac{x}{\pi}[\sin(t) - \cos(t)]$$

验证：$v(0,t) = \cos(t)$，$v(\pi,t) = \sin(t)$ ✓

#### Step 2：变换原问题

$$w(x,t) = u(x,t) - v(x,t)$$

计算 $v$ 的偏导数：
- $v_{tt} = -\cos(t) + \frac{x}{\pi}[-\sin(t) + \cos(t)]$
- $v_{xx} = 0$

源项：
$$F(x,t) = a^2 v_{xx} - v_{tt} = -[-\cos(t) + \frac{x}{\pi}(-\sin(t) + \cos(t))]$$

$$= \cos(t) + \frac{x}{\pi}(\sin(t) - \cos(t))$$

变换后的问题：
$$w_{tt} = w_{xx} + \cos(t) + \frac{x}{\pi}(\sin(t) - \cos(t))$$

边界条件：$w(0,t) = 0$，$w(\pi,t) = 0$ ✓

#### Step 3：初始条件

$$w(x,0) = u(x,0) - v(x,0) = \frac{x}{\pi} - [1 + \frac{x}{\pi}(0-1)] = \frac{x}{\pi} - 1 + \frac{x}{\pi} = \frac{2x}{\pi} - 1$$

$$w_t(x,0) = u_t(x,0) - v_t(x,0) = 0 - [-\sin(t) + \frac{x}{\pi}(\cos(t) + \sin(t))]\big|_{t=0}$$

$$= 0 - [0 + \frac{x}{\pi}(1+0)] = -\frac{x}{\pi}$$

#### Step 4-5：求解（步骤同前）

使用特征函数法，展开 $F(x,t)$ 和初始条件，求解时间方程。

最终解：
$$u(x,t) = w(x,t) + \cos(t) + \frac{x}{\pi}[\sin(t) - \cos(t)]$$

### 例题 6：热方程 - 导数边界条件

**题目**：
$$\begin{cases}
\frac{\partial u}{\partial t} = \frac{\partial^2 u}{\partial x^2}, & 0 < x < 1, \ t > 0 \\
\frac{\partial u}{\partial x}(0,t) = 1, \quad u(1,t) = 0, & t > 0 \\
u(x,0) = 0
\end{cases}$$

**解**：

#### Step 1：构造辅助函数

需要满足：$v_x(0,t) = 1$，$v(1,t) = 0$

选择二次函数：$v(x,t) = x - \frac{x^2}{2}$

验证：$v_x = 1 - x$，$v_x(0) = 1$，$v(1) = 1 - \frac{1}{2} = \frac{1}{2}$

不对！重新选择：$v(x,t) = x - \frac{x^2}{2}$，但 $v(1) = \frac{1}{2} \neq 0$

修正：$v(x,t) = x - \frac{x^2}{2}$ 不满足 $v(1,t) = 0$

正确选择：$v(x) = x - \frac{x^2}{2}$ 需要调整

更简单：设 $v(x,t) = A(t) + B(t)x + C(t)x^2$

边界条件：
- $v_x(0,t) = B(t) = 1$
- $v(1,t) = A(t) + B(t) + C(t) = 0$

选择 $A(t) = -1$，$B(t) = 1$，$C(t) = 0$：$v(x,t) = -1 + x$

验证：$v_x = 1$，$v_x(0) = 1$ ✓，$v(1) = 0$ ✓

$$v(x,t) = x - 1$$

#### Step 2：变换原问题

$$w(x,t) = u(x,t) - (x-1)$$

计算 $v$ 的偏导数：
- $v_t = 0$
- $v_{xx} = 0$

源项：
$$F(x,t) = c^2 v_{xx} - v_t = 0$$

变换后的问题：
$$w_t = w_{xx}$$

边界条件：$w_x(0,t) = 0$，$w(1,t) = 0$ ✓

#### Step 3：初始条件

$$w(x,0) = u(x,0) - v(x) = 0 - (x-1) = 1-x$$

#### Step 4：求解 $w$ 的问题

特征值问题：$X'' + \lambda X = 0$，$X'(0) = 0$，$X(1) = 0$

解：$\cos(\sqrt{\lambda}x)$，由 $X(1) = 0$ 得 $\sqrt{\lambda_n} = (n+\frac{1}{2})\pi$

特征值：$\lambda_n = (n+\frac{1}{2})^2\pi^2$，特征函数：$\cos((n+\frac{1}{2})\pi x)$

展开初始条件：
$$w(x,0) = 1-x = \sum_{n=0}^{\infty} A_n \cos((n+\frac{1}{2})\pi x)$$

$$A_n = 2\int_0^1 (1-x)\cos((n+\frac{1}{2})\pi x) \, dx = \frac{4(-1)^n}{(2n+1)^2\pi^2}$$

**通解**：
$$w(x,t) = \sum_{n=0}^{\infty} A_n e^{-(n+\frac{1}{2})^2\pi^2 t} \cos((n+\frac{1}{2})\pi x)$$

#### Step 5：最终解

$$\boxed{u(x,t) = (x-1) + \frac{4}{\pi^2} \sum_{n=0}^{\infty} \frac{(-1)^n}{(2n+1)^2} e^{-(n+\frac{1}{2})^2\pi^2 t} \cos((n+\frac{1}{2})\pi x)}$$

## 六、常见错误与注意事项

### ❌ 常见错误

1. **辅助函数不满足边界条件**
   - 必须验证 $v(0,t) = \alpha(t)$ 和 $v(L,t) = \beta(t)$

2. **变换后的源项计算错误**
   - $F(x,t) = a^2 v_{xx} - v_{tt}$（波动方程）
   - $F(x,t) = c^2 v_{xx} - v_t$（热方程）
   - 注意符号！

3. **初始条件变换错误**
   - $w(x,0) = u(x,0) - v(x,0)$
   - $w_t(x,0) = u_t(x,0) - v_t(x,0)$

4. **特征值问题边界条件**
   - 变换后 $w$ 的边界条件是**齐次**的
   - 根据原边界类型选择正确的特征函数

### ✅ 检查清单

- [ ] 辅助函数 $v(x,t)$ 是否满足边界条件？
- [ ] 变换后 $w$ 的边界条件是否为零？
- [ ] 源项 $F(x,t)$ 计算是否正确？
- [ ] 初始条件 $w(x,0)$ 和 $w_t(x,0)$ 是否正确？
- [ ] 特征值和特征函数是否对应齐次边界条件？
- [ ] 最终解 $u = w + v$ 是否满足原边界条件？

## 七、辅助函数构造速查表

| 边界条件类型 | 辅助函数 $v(x,t)$ |
|-------------|------------------|
| $u(0,t) = \alpha(t), \ u(L,t) = 0$ | $\alpha(t)(1 - \frac{x}{L})$ |
| $u(0,t) = 0, \ u(L,t) = \beta(t)$ | $\beta(t)\frac{x}{L}$ |

## 八、进阶技巧

### 1. 选择最优辅助函数

辅助函数的选择不唯一，但会影响求解难度：

**原则**：
- 优先选线性函数：$v = A(t) + B(t)x$
- 如果线性函数无法满足，再用二次函数
- 尽量让 $v_{xx} = 0$，这样源项 $F(x,t) = -v_{tt}$ 或 $-v_t$ 更简单

### 2. 处理复杂边界条件

如果边界条件复杂（如 $u(0,t) = \sin(t) + t^2$），可以：
- 选择 $v(x,t) = [\sin(t) + t^2](1 - \frac{x}{L})$
- 或者分解为多个简单辅助函数之和

### 3. 非齐次初始条件 + 非齐次边界

同时存在非齐次初始条件和非齐次边界条件时：
1. 先做边界齐次化：$w = u - v$
2. $w$ 的初始条件变为：$w(x,0) = \phi(x) - v(x,0)$
3. 继续用分离变量法求解

### 4. 验证最终解

解出 $u(x,t)$ 后，检查：
1. 是否满足原方程？
2. 是否满足边界条件？
3. 是否满足初始条件？
