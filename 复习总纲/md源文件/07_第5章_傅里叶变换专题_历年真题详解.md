## `<a id="知识总结"></a>` 一、核心知识点总结

### 1. 傅里叶变换定义

**傅里叶变换**（像函数）：
$$F(\alpha) = \mathcal{F}[f(x)] = \int_{-\infty}^{+\infty} f(x) e^{-i\alpha x} dx$$

**傅里叶逆变换**（像原函数）：
$$f(x) = \mathcal{F}^{-1}[F(\alpha)] = \frac{1}{2\pi} \int_{-\infty}^{+\infty} F(\alpha) e^{i\alpha x} d\alpha$$

---

### 2. 傅里叶变换核心性质（必考！）

| 性质名称 | 公式 | 重要程度 | 应用场景 |
|:--------:|:----:|:--------:|:---------|
| **线性性质** | $\mathcal{F}[c_1 f + c_2 g] = c_1 F(\alpha) + c_2 G(\alpha)$ | ⭐⭐⭐ | 分解复杂函数 |
| **位移性质** | $\mathcal{F}[f(x-c)] = e^{-i\alpha c} F(\alpha)$ | ⭐⭐ | 函数平移 |
| **微分性质** | $\mathcal{F}[f^{(n)}(x)] = (i\alpha)^n F(\alpha)$ | ⭐⭐⭐⭐⭐ | **核心！求解微分方程** |
| **卷积定理** | $\mathcal{F}[(f*g)(x)] = F(\alpha) G(\alpha)$ | ⭐⭐⭐⭐⭐ | **核心！求逆变换** |

> **卷积定义**：$(f * g)(x) = \int_{-\infty}^{+\infty} f(x-\xi) g(\xi) d\xi$

---

### 3. 解热传导方程的基本思路

对热传导方程 $u_t = a^2u_{xx} + \dots$，对 $x$ 作傅里叶变换：
- $u_t \to \frac{dU}{dt}$（变换与时间无关）
- $u_x \to i\alpha U(\alpha, t)$
- $u_{xx} \to (i\alpha)^2 U = -\alpha^2 U$

变换后得到关于 $t$ 的 **一阶常微分方程**，解出 $U(\alpha, t)$ 后再求逆变换。

---

### 4. 热传导方程基本解（必须背！）

$$\mathcal{F}^{-1}\left[e^{-c^2 \alpha^2 t}\right] = \frac{1}{2c\sqrt{\pi t}} e^{-\frac{x^2}{4c^2 t}}$$

---

### 5. 卷积定理的应用

齐次热传导方程初值问题解的泊松公式：
$$u(x, t) = \frac{1}{2a\sqrt{\pi t}} \int_{-\infty}^{+\infty} \phi(\xi) \exp\left(-\frac{(x-\xi)^2}{4a^2 t}\right) d\xi$$

### 🔵 【基础型】D4. 2015 第六题（12 分）

**题目**：$u_t = c^2u_{xx}$，$u(x,0) = x^2 + \cos x$。

**📖 知识点定位**：第五章 §5.2–§5.3 Fourier 变换法解热方程

---

#### 方法一：特解速查表法（考试推荐）

**解题思路**：
1. 初值是 $x^2 + \cos x$，拆为两部分
2. 查表直接写出解
3. 验证（可选，考试可省略）

**Step 1：查表得各部分解**

- **初值 $x^2$**：$v_1 = x^2 + 2c^2t$
- **初值 $\cos x$**：$v_2 = e^{-c^2t}\cos x$

**Step 2：叠加得最终解**

$$\boxed{u(x,t) = x^2 + 2c^2t + e^{-c^2t}\cos x}$$

---

#### 方法二：完整傅里叶变换法（学习理解用）

**解题思路**：
1. 对 $x$ 作 Fourier 变换
2. 解关于 $t$ 的 ODE
3. 求逆变换（利用性质，不用算积分）

**Step 1：对 $x$ 作 Fourier 变换**

设 $U(\alpha,t) = \mathcal{F}[u(x,t)]$，$\Phi(\alpha) = \mathcal{F}[x^2 + \cos x]$。

对方程两边作变换：
$$\frac{dU}{dt} = -c^2\alpha^2 U$$

对初值作变换：
$$U(\alpha,0) = \Phi(\alpha) = \mathcal{F}[x^2] + \mathcal{F}[\cos x]$$

**Step 2：解 ODE**

这是一阶线性 ODE，通解为：
$$U(\alpha,t) = \Phi(\alpha) e^{-c^2\alpha^2 t}$$

代入 $\Phi(\alpha)$：
$$U(\alpha,t) = \mathcal{F}[x^2] \cdot e^{-c^2\alpha^2 t} + \mathcal{F}[\cos x] \cdot e^{-c^2\alpha^2 t}$$

**Step 3：求逆变换（利用性质）**

利用微分性质的逆用：$\mathcal{F}[x^n f(x)] = i^n F^{(n)}(\alpha)$，但更简单的是**利用卷积定理**。

或者直接利用**基本解的性质**：
- $e^{-c^2\alpha^2 t}$ 的逆变换是热核 $K(x,t) = \frac{1}{2c\sqrt{\pi t}} e^{-\frac{x^2}{4c^2 t}}$

因此：
$$u(x,t) = (x^2 + \cos x) * K(x,t)$$

但对于 $x^2$ 和 $\cos x$，我们可以用**傅里叶变换的微分性质逆推**：

**对于 $x^2$ 部分**：
设 $u_1(x,t)$ 对应初值 $x^2$，则 $U_1(\alpha,t) = \mathcal{F}[x^2] e^{-c^2\alpha^2 t}$

利用性质 $\mathcal{F}[x^2] = 2\pi i^2 \delta''(\alpha) = -2\pi \delta''(\alpha)$（广义函数意义下）

逆变换：
$$u_1(x,t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} (-2\pi \delta''(\alpha)) e^{-c^2\alpha^2 t} e^{i\alpha x} d\alpha$$
$$= -\int_{-\infty}^{\infty} \delta''(\alpha) e^{-c^2\alpha^2 t} e^{i\alpha x} d\alpha$$
$$= -\frac{d^2}{d\alpha^2} \left[e^{-c^2\alpha^2 t} e^{i\alpha x}\right] \bigg|_{\alpha=0}$$
$$= -\frac{d^2}{d\alpha^2} \left[e^{-c^2\alpha^2 t + i\alpha x}\right] \bigg|_{\alpha=0}$$

求导：
一阶导：$(-2c^2\alpha t + ix) e^{-c^2\alpha^2 t + i\alpha x}$
二阶导：$(-2c^2 t + (-2c^2\alpha t + ix)^2) e^{-c^2\alpha^2 t + i\alpha x}$

代入 $\alpha=0$：
$$= -[-2c^2 t + (ix)^2] = -[-2c^2 t - x^2] = x^2 + 2c^2 t$$

**对于 $\cos x$ 部分**：
设 $u_2(x,t)$ 对应初值 $\cos x$

$\cos x = \frac{e^{ix} + e^{-ix}}{2}$，其傅里叶变换为：
$$\mathcal{F}[\cos x] = \pi [\delta(\alpha-1) + \delta(\alpha+1)]$$

因此：
$$U_2(\alpha,t) = \pi [\delta(\alpha-1) + \delta(\alpha+1)] e^{-c^2\alpha^2 t}$$

逆变换：
$$u_2(x,t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} \pi [\delta(\alpha-1) + \delta(\alpha+1)] e^{-c^2\alpha^2 t} e^{i\alpha x} d\alpha$$
$$= \frac{1}{2} [e^{-c^2 \cdot 1^2 t} e^{i\cdot1\cdot x} + e^{-c^2 \cdot (-1)^2 t} e^{i\cdot(-1)\cdot x}]$$
$$= \frac{1}{2} e^{-c^2 t} (e^{ix} + e^{-ix}) = e^{-c^2 t} \cos x$$

**Step 4：叠加得最终解**

$$u(x,t) = u_1(x,t) + u_2(x,t) = x^2 + 2c^2t + e^{-c^2t}\cos x$$

与特解法结果完全一致！

$$\boxed{u(x,t) = x^2 + 2c^2t + e^{-c^2t}\cos x}$$

### 🔵 【基础型】D5. 2014 第六题（12 分）

**题目**：$u_t = c^2u_{xx}$，$u(x,0) = x^2 + \sin 2x$。

**📖 知识点定位**：第五章 §5.2–§5.3 Fourier 变换法解热方程

---

#### 方法一：特解速查表法（考试推荐）

**解题思路**：
- 拆为 $x^2$ 和 $\sin 2x$ 两部分
- 注意 $\sin 2x$ 中 $\omega = 2$，所以衰减系数是 $-c^2 \cdot 2^2 = -4c^2$

**Step 1：查表得各部分解**

- **初值 $x^2$**：$v_1 = x^2 + 2c^2t$
- **初值 $\sin 2x$**：$v_2 = e^{-4c^2t}\sin 2x$

**Step 2：叠加得最终解**

$$\boxed{u(x,t) = x^2 + 2c^2t + e^{-4c^2t}\sin 2x}$$

---

#### 方法二：完整傅里叶变换法（学习理解用）

**解题思路**：
1. 对 $x$ 作 Fourier 变换
2. 解关于 $t$ 的 ODE
3. 求逆变换（利用 $\delta$ 函数筛选性质）

**Step 1：对 $x$ 作 Fourier 变换**

设 $U(\alpha,t) = \mathcal{F}[u(x,t)]$，$\Phi(\alpha) = \mathcal{F}[x^2 + \sin 2x]$。

方程变换后：
$$\frac{dU}{dt} = -c^2\alpha^2 U$$

解为：
$$U(\alpha,t) = \Phi(\alpha) e^{-c^2\alpha^2 t}$$

**Step 2：初值的傅里叶变换**

$\sin 2x = \frac{e^{i2x} - e^{-i2x}}{2i}$，其傅里叶变换为：
$$\mathcal{F}[\sin 2x] = \frac{\pi}{i} [\delta(\alpha-2) - \delta(\alpha+2)]$$

$\mathcal{F}[x^2] = -2\pi \delta''(\alpha)$（同 D4）

**Step 3：逆变换求 $x^2$ 部分**

同 D4，直接写出：
$$u_1(x,t) = x^2 + 2c^2t$$

**Step 4：逆变换求 $\sin 2x$ 部分**

设 $u_2(x,t)$ 对应初值 $\sin 2x$：
$$U_2(\alpha,t) = \frac{\pi}{i} [\delta(\alpha-2) - \delta(\alpha+2)] e^{-c^2\alpha^2 t}$$

逆变换：
$$u_2(x,t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} \frac{\pi}{i} [\delta(\alpha-2) - \delta(\alpha+2)] e^{-c^2\alpha^2 t} e^{i\alpha x} d\alpha$$
$$= \frac{1}{2i} [e^{-c^2 \cdot 2^2 t} e^{i\cdot2\cdot x} - e^{-c^2 \cdot (-2)^2 t} e^{i\cdot(-2)\cdot x}]$$
$$= \frac{1}{2i} e^{-4c^2 t} (e^{i2x} - e^{-i2x}) = e^{-4c^2 t} \sin 2x$$

**Step 5：叠加得最终解**

$$\boxed{u(x,t) = x^2 + 2c^2t + e^{-4c^2t}\sin 2x}$$

### 🔵 【基础型】D6. 2013 第六题（10 分）

**题目**：$u_t = c^2u_{xx}$，$u(x,0) = 1 + \cos x$。

**📖 知识点定位**：第五章 §5.2–§5.3 Fourier 变换法解热方程

---

#### 方法一：特解速查表法（考试推荐）

**解题思路**：
- 拆为常数 1 和 $\cos x$ 两部分
- 常数是热方程的稳态解（导数为 0）

**Step 1：查表得各部分解**

- **初值 $1$**：$v_1 = 1$（常数不变）
- **初值 $\cos x$**：$v_2 = e^{-c^2t}\cos x$

**Step 2：叠加得最终解**

$$\boxed{u(x,t) = 1 + e^{-c^2t}\cos x}$$

---

#### 方法二：完整傅里叶变换法（学习理解用）

**解题思路**：
1. 对 $x$ 作 Fourier 变换
2. 解关于 $t$ 的 ODE
3. 求逆变换（常数对应 $\delta$ 函数）

**Step 1：对 $x$ 作 Fourier 变换**

方程变换后：
$$\frac{dU}{dt} = -c^2\alpha^2 U, \quad U(\alpha,0) = \Phi(\alpha) = \mathcal{F}[1] + \mathcal{F}[\cos x]$$

**Step 2：初值的傅里叶变换**

$\mathcal{F}[1] = 2\pi \delta(\alpha)$（常数的傅里叶变换是 $\delta$ 函数）
$\mathcal{F}[\cos x] = \pi [\delta(\alpha-1) + \delta(\alpha+1)]$

**Step 3：解 ODE**

$$U(\alpha,t) = [2\pi \delta(\alpha) + \pi \delta(\alpha-1) + \pi \delta(\alpha+1)] e^{-c^2\alpha^2 t}$$

**Step 4：求逆变换**

$$u(x,t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} [2\pi \delta(\alpha) + \pi \delta(\alpha-1) + \pi \delta(\alpha+1)] e^{-c^2\alpha^2 t} e^{i\alpha x} d\alpha$$

分项计算：
- 第一项：$\frac{1}{2\pi} \int 2\pi \delta(\alpha) e^{0} e^{i\alpha x} d\alpha = 1$
- 第二项：$\frac{1}{2\pi} \int \pi \delta(\alpha-1) e^{-c^2 t} e^{i x} d\alpha = \frac{1}{2} e^{-c^2 t} e^{i x}$
- 第三项：$\frac{1}{2\pi} \int \pi \delta(\alpha+1) e^{-c^2 t} e^{-i x} d\alpha = \frac{1}{2} e^{-c^2 t} e^{-i x}$

合并后两项：
$$\frac{1}{2} e^{-c^2 t} (e^{i x} + e^{-i x}) = e^{-c^2 t} \cos x$$

**Step 5：叠加得最终解**

$$\boxed{u(x,t) = 1 + e^{-c^2t}\cos x}$$

> 💡 **母题发现**：2013-2015 三道题本质是同一道母题用不同初值！都是 $u_t = c^2u_{xx}$，只是初值不同。

### 🟡 【加源项】D1. 2020 第六题（12 分）

**题目**：$u_t = a^2u_{xx} + t^2$，$u(x,0) = x^2 + \sin x$。

**📖 知识点定位**：第五章 §5.3 Fourier 变换解定解问题

---

#### 方法一：叠加原理分离法（考试推荐）

**解题思路**：
1. 源项 $t^2$ 与 $x$ 无关，利用叠加原理分离
2. 令 $u = v + w(t)$，$v$ 满足齐次方程，$w$ 只含 $t$
3. $w(t)$ 直接积分：$w' = t^2 \implies w = t^3/3$
4. $v$ 的初值不变，用特解凑法

**Step 1：叠加原理分离源项**

令 $u(x,t) = v(x,t) + w(t)$，则 $w_t = t^2 \implies w(t) = \frac{t^3}{3}$

$v$ 满足齐次热传导方程：
$$v_t = a^2v_{xx}, \quad v(x,0) = x^2 + \sin x$$

**Step 2：用特解凑法**

- 初值 $x^2$：$v_1 = x^2 + 2a^2t$
- 初值 $\sin x$：$v_2 = e^{-a^2t}\sin x$

**Step 3：合成最终解**

$$\boxed{u(x,t) = x^2 + 2a^2t + \frac{t^3}{3} + e^{-a^2t}\sin x}$$

> ✅ **验证**：$u_t = 2a^2 + t^2 - a^2 e^{-a^2t}\sin x$，$a^2u_{xx} = 2a^2 - a^2 e^{-a^2t}\sin x$，加 $t^2$ 匹配 ✓

---

#### 方法二：完整傅里叶变换法（学习理解用）

**解题思路**：
1. 对 $x$ 作 Fourier 变换，源项也作变换（$t^2$ 与 $x$ 无关，变换后仍为 $t^2 \cdot 2\pi \delta(\alpha)$）
2. 解关于 $t$ 的非齐次线性 ODE
3. 求逆变换

**Step 1：对 $x$ 作 Fourier 变换**

设 $U(\alpha,t) = \mathcal{F}[u(x,t)]$，$\Phi(\alpha) = \mathcal{F}[x^2 + \sin x]$

对方程两边变换：
$$\frac{dU}{dt} = -a^2\alpha^2 U + F(\alpha,t)$$

其中源项 $F(\alpha,t) = \mathcal{F}[t^2] = t^2 \cdot \mathcal{F}[1] = 2\pi t^2 \delta(\alpha)$

初值：$U(\alpha,0) = \Phi(\alpha)$

**Step 2：解非齐次线性 ODE**

一阶线性 ODE 的标准形式：$\frac{dU}{dt} + P(t)U = Q(t)$

这里 $P(t) = a^2\alpha^2$，$Q(t) = 2\pi t^2 \delta(\alpha)$

积分因子：$\mu(t) = e^{\int a^2\alpha^2 dt} = e^{a^2\alpha^2 t}$

通解：
$$U(\alpha,t) = e^{-a^2\alpha^2 t} \left[\Phi(\alpha) + \int_0^t 2\pi \tau^2 \delta(\alpha) e^{a^2\alpha^2 \tau} d\tau\right]$$

**Step 3：分情况讨论 $\alpha$**

**情况1：$\alpha \neq 0$**
此时 $\delta(\alpha) = 0$，积分项消失：
$$U(\alpha,t) = \Phi(\alpha) e^{-a^2\alpha^2 t} \quad (\alpha \neq 0)$$

**情况2：$\alpha = 0$**
此时 $\delta(\alpha) = \infty$，单独计算：
$$U(0,t) = e^{0} \left[\Phi(0) + \int_0^t 2\pi \tau^2 \cdot \infty \cdot e^{0} d\tau\right]$$

更简单的方法：回到原方程，当 $\alpha=0$ 时，变换后的方程为：
$$\frac{dU}{dt}(0,t) = 0 + t^2 \cdot 2\pi \delta(0)? \quad \text{不对，换一种思路。}$$

**更好的方法：直接用卷积定理求解非齐次方程**

回忆教材中，非齐次热传导方程初值问题的解为：
$$u(x,t) = \mathcal{F}^{-1}\left[\Phi(\alpha)e^{-a^2\alpha^2 t}\right] + \int_0^t \mathcal{F}^{-1}\left[F(\alpha,\tau)e^{-a^2\alpha^2(t-\tau)}\right] d\tau$$

这里 $F(\alpha,\tau) = \mathcal{F}[\tau^2] = 2\pi \tau^2 \delta(\alpha)$

**计算第一项（齐次解）**：
同前几道题，对应初值 $x^2 + \sin x$：
$$\mathcal{F}^{-1}\left[\Phi(\alpha)e^{-a^2\alpha^2 t}\right] = x^2 + 2a^2t + e^{-a^2t}\sin x$$

**计算第二项（特解）**：
$$\int_0^t \mathcal{F}^{-1}\left[2\pi \tau^2 \delta(\alpha) e^{-a^2\alpha^2(t-\tau)}\right] d\tau$$
$$= \int_0^t \frac{1}{2\pi} \int_{-\infty}^{\infty} 2\pi \tau^2 \delta(\alpha) e^{-a^2\alpha^2(t-\tau)} e^{i\alpha x} d\alpha d\tau$$
$$= \int_0^t \tau^2 \cdot e^{0} \cdot e^{0} d\tau = \int_0^t \tau^2 d\tau = \frac{t^3}{3}$$

**Step 4：叠加得最终解**

$$\boxed{u(x,t) = x^2 + 2a^2t + \frac{t^3}{3} + e^{-a^2t}\sin x}$$

### 🟡 【加反应项】D2. 2019 第六题（12 分）

**题目**：$u_t = a^2u_{xx} + cu$，$u(x,0) = x^2 + \cos x$。

**📖 知识点定位**：第五章 §5.3 Fourier 变换解热传导方程

---

#### 方法一：指数变换法（考试推荐）

**解题思路**：
1. 反应项 $cu$ 可通过指数变换消去
2. 令 $v = e^{-ct}u$，则 $u = e^{ct}v$
3. 代入得 $v_t = a^2v_{xx}$（齐次！）
4. 解 $v$ 后回代

**Step 1：指数变换消去反应项**

令 $v(x,t) = e^{-ct}u(x,t)$，则：
$$v_t = e^{-ct}(u_t - cu) = e^{-ct}(a^2u_{xx}) = a^2v_{xx}$$

初始条件：$v(x,0) = x^2 + \cos x$

**Step 2：解齐次方程**

- 初值 $x^2$：$v_1 = x^2 + 2a^2t$
- 初值 $\cos x$：$v_2 = e^{-a^2t}\cos x$

**Step 3：回代 $u = e^{ct}v$**

$$\boxed{u(x,t) = e^{ct}(x^2 + 2a^2t + e^{-a^2t}\cos x)}$$

> ✅ **验证**：$u_t = cu + e^{ct}(2a^2 - a^2 e^{-a^2t}\cos x)$，$a^2u_{xx} = a^2 e^{ct}(2 - e^{-a^2t}\cos x)$，匹配 ✓

---

#### 方法二：完整傅里叶变换法（学习理解用）

**解题思路**：
1. 直接对 $x$ 作 Fourier 变换
2. 反应项 $cu$ 变换后为 $cU$
3. 解关于 $t$ 的 ODE
4. 求逆变换

**Step 1：对 $x$ 作 Fourier 变换**

设 $U(\alpha,t) = \mathcal{F}[u(x,t)]$，$\Phi(\alpha) = \mathcal{F}[x^2 + \cos x]$

对方程两边变换：
$$\frac{dU}{dt} = -a^2\alpha^2 U + cU = (-a^2\alpha^2 + c)U$$

初值：$U(\alpha,0) = \Phi(\alpha)$

**Step 2：解 ODE**

这是一阶线性齐次 ODE，解为：
$$U(\alpha,t) = \Phi(\alpha) e^{(-a^2\alpha^2 + c)t} = e^{ct} \cdot \Phi(\alpha) e^{-a^2\alpha^2 t}$$

**Step 3：求逆变换**

利用傅里叶变换的线性性质：
$$u(x,t) = e^{ct} \cdot \mathcal{F}^{-1}\left[\Phi(\alpha) e^{-a^2\alpha^2 t}\right]$$

括号内就是齐次热传导方程初值为 $x^2 + \cos x$ 的解（同 D4）：
$$\mathcal{F}^{-1}\left[\Phi(\alpha) e^{-a^2\alpha^2 t}\right] = x^2 + 2a^2t + e^{-a^2t}\cos x$$

**Step 4：乘以 $e^{ct}$ 得最终解**

$$\boxed{u(x,t) = e^{ct}(x^2 + 2a^2t + e^{-a^2t}\cos x)}$$

或者写成：
$$\boxed{u(x,t) = e^{ct}(x^2 + 2a^2t) + e^{(c-a^2)t}\cos x}$$

### 🟠 【加对流-反应项】D3. 2018 第七题（5 分）

**题目**：$u_t = a^2u_{xx} + bu_x + cu$，$u(x,0) = \varphi(x)$。求一般解和 $\varphi(x)=\sin x$ 的特解。

**📖 知识点定位**：第五章 §5.3 Fourier 变换法

---

#### 方法一：完整傅里叶变换法（本题唯一方法）

**解题思路**：
1. 直接对 $x$ 作 Fourier 变换
2. $u_{xx} \to -\alpha^2 U$，$u_x \to i\alpha U$
3. 解 ODE 得 $U(\alpha, t) = \Phi(\alpha) e^{(-a^2\alpha^2 + ib\alpha + c)t}$
4. 逆变换：$e^{ib\alpha t}$ 是平移 $x \to x+bt$，$e^{ct}$ 是指数因子，$e^{-a^2\alpha^2t}$ 是热核

**Step 1：对 $x$ 做 Fourier 变换**

设 $U(\alpha,t) = \mathcal{F}[u(x,t)]$，$\Phi(\alpha) = \mathcal{F}[\varphi(x)]$

对方程两边作变换：
- $u_t \to \frac{dU}{dt}$
- $u_{xx} \to (i\alpha)^2 U = -\alpha^2 U$
- $u_x \to i\alpha U$
- $cu \to cU$

得到：
$$\frac{dU}{dt} = (-a^2\alpha^2 + ib\alpha + c)U$$

**Step 2：解 ODE**

这是一阶线性齐次 ODE，解为：
$$U(\alpha,t) = \Phi(\alpha) e^{(-a^2\alpha^2 + ib\alpha + c)t}$$

整理指数：
$$U(\alpha,t) = e^{ct} \cdot e^{ib\alpha t} \cdot \Phi(\alpha) \cdot e^{-a^2\alpha^2 t}$$

**Step 3：求逆变换得一般解**

利用傅里叶变换的性质逐项求逆：

1. **$e^{ct}$**：常数因子，直接提出
2. **$e^{ib\alpha t}$**：位移性质，逆变换为 $\delta(x+bt)$（或直接理解为平移 $x \to x+bt$）
3. **$e^{-a^2\alpha^2 t}$**：热核，逆变换为 $\frac{1}{2a\sqrt{\pi t}} e^{-\frac{x^2}{4a^2 t}}$

利用卷积定理，逆变换为：
$$u(x,t) = e^{ct} \cdot \left[\varphi(x+bt) * \frac{1}{2a\sqrt{\pi t}} e^{-\frac{x^2}{4a^2 t}}\right]$$

写出卷积积分：

$$\boxed{u(x,t) = \frac{e^{ct}}{2a\sqrt{\pi t}}\int_{-\infty}^{\infty}\varphi(\xi)\exp\!\left(-\frac{(x+bt-\xi)^2}{4a^2t}\right)d\xi}$$

**Step 4：当 $\varphi(x)=\sin x$ 时求特解**

先求 $\sin x$ 的 Fourier 变换：
$$\sin x = \frac{e^{ix} - e^{-ix}}{2i}$$
$$\mathcal{F}[\sin x] = \frac{\pi}{i} [\delta(\alpha-1) - \delta(\alpha+1)]$$

代入 $U(\alpha,t)$：
$$U(\alpha,t) = \frac{\pi}{i} [\delta(\alpha-1) - \delta(\alpha+1)] e^{(-a^2\alpha^2 + ib\alpha + c)t}$$

求逆变换：
$$u(x,t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} \frac{\pi}{i} [\delta(\alpha-1) - \delta(\alpha+1)] e^{(-a^2\alpha^2 + ib\alpha + c)t} e^{i\alpha x} d\alpha$$
$$= \frac{1}{2i} \left[e^{(-a^2\cdot1^2 + ib\cdot1 + c)t} e^{i\cdot1\cdot x} - e^{(-a^2\cdot(-1)^2 + ib\cdot(-1) + c)t} e^{i\cdot(-1)\cdot x}\right]$$
$$= \frac{1}{2i} e^{(c-a^2)t} \left[e^{i(bt+x)} - e^{-i(bt+x)}\right]$$
$$= e^{(c-a^2)t} \sin(x+bt)$$

$$\boxed{u(x,t) = e^{(c-a^2)t}\sin(x+bt)}$$

> 🏆 **物理含义**：
> - $e^{-a^2t}$ = 扩散衰减
> - $\sin(x+bt)$ = 对流平移（以速度 $-b$ 移动）
> - $e^{ct}$ = 反应增长/衰减

---

#### 方法二：补充——指数变换 + 平移变换（可选拓展）

**Step 1：先消去反应项**

令 $v = e^{-ct}u$，则 $u = e^{ct}v$，代入得：
$$v_t = a^2v_{xx} + bv_x$$

**Step 2：再消去对流项**

令 $w(x,t) = v(x+bt, t)$（即沿特征线平移），则：
$$w_t = a^2w_{xx}$$

**Step 3：解齐次热方程**

$$w(x,t) = \frac{1}{2a\sqrt{\pi t}} \int_{-\infty}^{\infty} \varphi(\xi) e^{-\frac{(x-\xi)^2}{4a^2 t}} d\xi$$

**Step 4：回代得解**

$$v(x,t) = w(x-bt, t) = \frac{1}{2a\sqrt{\pi t}} \int_{-\infty}^{\infty} \varphi(\xi) e^{-\frac{(x-bt-\xi)^2}{4a^2 t}} d\xi$$
$$u(x,t) = e^{ct}v(x,t) = \frac{e^{ct}}{2a\sqrt{\pi t}} \int_{-\infty}^{\infty} \varphi(\xi) e^{-\frac{(x+bt-\xi)^2}{4a^2 t}} d\xi$$

（与傅里叶变换法结果完全一致！）

## `<a id="变体"></a>` 四、题型变体规律总结

### 傅里叶变换热传导方程变体谱系图

```
基础型：u_t = a²u_xx
│
├── 初值变体：φ(x) 变化
│   ├── 常数 + 三角函数：D6 (1 + cosx)
│   ├── 二次函数 + 正弦：D5 (x² + sin2x)
│   └── 二次函数 + 余弦：D4 (x² + cosx)
│
├── 加源项：u_t = a²u_xx + f(t)
│   └── D1：f(t) = t²，与 x 无关
│
├── 加反应项：u_t = a²u_xx + cu
│   └── D2：指数变换 v = e^(-ct)u
│
└── 加对流-反应项：u_t = a²u_xx + bu_x + cu
    └── D3：Fourier 变换直接解，卷积定积分
```

## `<a id="模板"></a>` 五、解题通用模板

### 模板 A：齐次型（仅初值变化）

**题目形式**：$u_t = c^2u_{xx}$，$u(x,0) = \varphi(x)$

**解题步骤**：

```
第1步：分解初值
   φ(x) = 多项式项 + 三角函数项
   
第2步：查表写出各分量解
   - 常数 C → C
   - x² → x² + 2c²t
   - cosωx → e^(-c²ω²t)cosωx
   - sinωx → e^(-c²ω²t)sinωx
   
第3步：叠加
   u = Σ各分量解
   
第4步：验证（可选）
   求 u_t 和 c²u_xx 是否相等
```

---

### 模板 B：加源项型

**题目形式**：$u_t = a^2u_{xx} + f(t)$

**解题步骤**：

```
第1步：分离变量
   令 u = v(x,t) + w(t)
   
第2步：求 w(t)
   w' = f(t) → 积分求 w(t)
   
第3步：v 满足齐次方程，按模板A解
   
第4步：合成 u = v + w
```

---

### 模板 C：加反应项型

**题目形式**：$u_t = a^2u_{xx} + cu$

**解题步骤**：

```
第1步：指数变换消去 cu
   令 v = e^(-ct)u，则 v_t = a²v_xx
   
第2步：按模板A解 v
   
第3步：回代 u = e^(ct)v
```

---

### 模板 D：加对流-反应项（一般解）

**题目形式**：$u_t = a^2u_{xx} + bu_x + cu$

**解题步骤**：

```
第1步：对 x 作 Fourier 变换
   U_t = (-a²α² + ibα + c)U
   
第2步：解 ODE
   U(α,t) = Φ(α)e^(-a²α²+ibα+c)t
   
第3步：逆变换（卷积定理）
   - e^(ibαt) → 平移 x → x+bt
   - e^(-a²α²t) → 热核
   - e^(ct) → 指数因子
   
第4步：写出卷积积分
   u = 热核 * φ(x+bt)
```

## ⚠️ 常见易错点提醒

1. **三角函数的频率别搞错**！$\sin 2x$ 的衰减是 $e^{-4c^2t}$，不是 $e^{-c^2t}$

2. **加反应项最后别忘了乘 $e^{ct}$**！很多同学解完 $v$ 就忘了回代

3. **对流项是平移 $x \to x+bt$**！注意符号：$bu_x$ 对应正号平移

4. **初始条件别带错**！分离变量后 $v$ 的初始条件和 $u$ 相同
