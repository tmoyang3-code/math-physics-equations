## 一、傅里叶变换填空题

### 题1：分段函数的傅里叶变换（2023级）

**题目**：设
$$f(x) = \begin{cases} e^{ax}, & |x| \leq a, \\ 0, & |x| > a, \end{cases}$$
则 $\mathcal{F}[f(x)] =$ ______.

**解答**：
按本课程常用约定
$$\mathcal{F}[f](\alpha) = \int_{-\infty}^{+\infty} f(x)e^{-i\alpha x}dx,$$
由于 $f(x)$ 在 $[-a,a]$ 外为 0，故
$$\mathcal{F}[f](\alpha) = \int_{-a}^{a} e^{ax}e^{-i\alpha x}dx = \int_{-a}^{a} e^{(a-i\alpha)x}dx.$$
直接积分：
$$\int_{-a}^{a} e^{(a-i\alpha)x}dx = \frac{1}{a-i\alpha}\left[e^{(a-i\alpha)x}\right]_{-a}^{a} = \frac{e^{(a-i\alpha)a} - e^{-(a-i\alpha)a}}{a-i\alpha}.$$
化简：
$$\mathcal{F}[f](\alpha) = \frac{e^{a^2-i\alpha a} - e^{-a^2+i\alpha a}}{a-i\alpha}.$$
也可写成双曲正弦形式：
$$\mathcal{F}[f](\alpha) = \frac{2\sinh(a^2-i\alpha a)}{a-i\alpha}.$$

**答案**：
$$\boxed{\frac{e^{a^2-i\alpha a} - e^{-a^2+i\alpha a}}{a-i\alpha}}$$

---

### 题2：傅里叶卷积定理（2023级）

**题目**：设
$$\mathcal{F}[f(x)] = F(\alpha), \qquad \mathcal{F}[\sin x] = G(\alpha),$$
则
$$\mathcal{F}^{-1}[F(\alpha)G(\alpha)] = \underline{\qquad}.$$

**解答**：
在本课程约定下，**傅里叶卷积定理**为：
$$\mathcal{F}[f * g] = \mathcal{F}[f] \cdot \mathcal{F}[g],$$
其中卷积 $(f*g)(x) = \int_{-\infty}^{+\infty} f(\xi)g(x-\xi)d\xi$.

取逆变换即得：
$$\mathcal{F}^{-1}[F(\alpha)G(\alpha)] = (f * \sin)(x).$$
展开：
$$\mathcal{F}^{-1}[F(\alpha)G(\alpha)] = \int_{-\infty}^{+\infty} f(\xi)\sin(x-\xi)d\xi.$$

**答案**：
$$\boxed{(f * \sin)(x) = \int_{-\infty}^{+\infty} f(\xi)\sin(x-\xi)d\xi}$$

---

### 题3：热传导方程的傅里叶变换（2020级）

**题目**：设函数 $u = u(x,t)$ 的傅里叶变换为 $U(\alpha,t)$，则热传导方程
$$u_t = c^2 u_{xx} + be^{-|x|}$$
的傅里叶变换式为 ______.

**解答**：
对 $x$ 取傅里叶变换（$t$ 视为参数），利用微分性质：
- $\mathcal{F}[u_t] = U_t(\alpha,t)$（变换与 $t$ 无关）
- $\mathcal{F}[u_{xx}] = (i\alpha)^2 U = -\alpha^2 U(\alpha,t)$

对非齐次项，查变换表：
$$\mathcal{F}[e^{-a|x|}] = \frac{2a}{a^2+\alpha^2},$$
取 $a=1$，得
$$\mathcal{F}[be^{-|x|}] = \frac{2b}{1+\alpha^2}.$$

代入方程：
$$U_t(\alpha,t) = -c^2\alpha^2 U(\alpha,t) + \frac{2b}{1+\alpha^2}.$$

**答案**：
$$\boxed{U_t(\alpha,t) = -c^2\alpha^2 U(\alpha,t) + \frac{2b}{1+\alpha^2}}$$

### 题4：拉普拉斯方程的傅里叶变换（2023级）

**题目**：在拉普拉斯方程
$$u_{xx} + u_{yy} = 0$$
两边关于 $x$ 做傅里叶变换，得到常微分方程是 ______.

**解答**：
设
$$U(\alpha,y) = \mathcal{F}_x[u(x,y)],$$
其中下标表示仅对 $x$ 做变换，$y$ 视为参数。

利用微分性质：
- $\mathcal{F}[u_{xx}] = (i\alpha)^2 U = -\alpha^2 U$
- $\mathcal{F}[u_{yy}] = U_{yy}$（变换与 $y$ 无关）

代入方程：
$$-\alpha^2 U + U_{yy} = 0,$$
整理为标准二阶常微分方程：
$$U_{yy} - \alpha^2 U = 0.$$

**答案**：
$$\boxed{U_{yy} - \alpha^2 U = 0}$$

## 二、拉普拉斯变换填空题

### 题1：指数衰减余弦的拉普拉斯变换（2023级）

**题目**：$\mathcal{L}[e^{-at}\cos bt] =$ ______.

**解答**：
首先回忆基本变换公式：
$$\mathcal{L}[\cos bt] = \frac{s}{s^2+b^2}.$$

再利用**位移性质**（s-shifting）：
若 $\mathcal{L}[f(t)] = F(s)$，则
$$\mathcal{L}[e^{-at}f(t)] = F(s+a).$$

因此：
$$\mathcal{L}[e^{-at}\cos bt] = \frac{(s+a)}{(s+a)^2+b^2}.$$

**答案**：
$$\boxed{\frac{s+a}{(s+a)^2+b^2}}$$

---

### 题2：含乘 t、指数、正弦的拉普拉斯变换（2020级）

**题目**：函数 $f(t) = te^{at}\sin\omega t$ 的拉普拉斯变换为 ______.

**解答**：
从基本公式出发，分三步：

**Step 1**：正弦函数变换
$$\mathcal{L}[\sin\omega t] = \frac{\omega}{s^2+\omega^2}.$$

**Step 2**：应用位移性质（s-shifting）
$$\mathcal{L}[e^{at}\sin\omega t] = \frac{\omega}{(s-a)^2+\omega^2}.$$

**Step 3**：应用乘 t 性质
若 $\mathcal{L}[f(t)] = F(s)$，则
$$\mathcal{L}[t f(t)] = -\frac{d}{ds}F(s).$$

对 Step 2 的结果求导：
$$\mathcal{L}[te^{at}\sin\omega t] = -\frac{d}{ds}\left[\frac{\omega}{(s-a)^2+\omega^2}\right].$$
求导：
$$-\omega \cdot \frac{-2(s-a)}{[(s-a)^2+\omega^2]^2} = \frac{2\omega(s-a)}{[(s-a)^2+\omega^2]^2}.$$

**答案**：
$$\boxed{\frac{2\omega(s-a)}{[(s-a)^2+\omega^2]^2}}$$

### 题3：拉普拉斯逆变换（部分分式分解）（2023级）

**题目**：$\mathcal{L}^{-1}\left[\frac{1}{s(s^2+1)}\right] =$ ______.

**解答**：
对被变换函数做部分分式分解：
$$\frac{1}{s(s^2+1)} = \frac{A}{s} + \frac{Bs+C}{s^2+1}.$$

通分比较分子：
$$1 = A(s^2+1) + (Bs+C)s.$$

代入 $s=0$：
$$1 = A \cdot 1 \Rightarrow A = 1.$$

比较 $s^2$ 系数：
$$0 = A + B \Rightarrow B = -1.$$

比较 $s$ 系数：
$$0 = C.$$

因此：
$$\frac{1}{s(s^2+1)} = \frac{1}{s} - \frac{s}{s^2+1}.$$

分别取逆变换：
$$\mathcal{L}^{-1}\left[\frac{1}{s}\right] = 1,$$
$$\mathcal{L}^{-1}\left[\frac{s}{s^2+1}\right] = \cos t.$$

合并得：
$$\mathcal{L}^{-1}\left[\frac{1}{s(s^2+1)}\right] = 1 - \cos t.$$

**答案**：
$$\boxed{1 - \cos t}$$

### 题4：拉普拉斯逆变换（三重极点+单极点）（2020级）

**题目**：函数 $G(s) = \frac{1}{s^2(s+1)(s+2)}$ 的拉普拉斯逆变换为 ______.

**解答**：
对 $G(s)$ 做部分分式分解，设：
$$\frac{1}{s^2(s+1)(s+2)} = \frac{A}{s} + \frac{B}{s^2} + \frac{C}{s+1} + \frac{D}{s+2}.$$

通分比较分子：
$$1 = As(s+1)(s+2) + B(s+1)(s+2) + Cs^2(s+2) + Ds^2(s+1).$$

**确定系数**：
- $s = 0$：$1 = B \cdot 1 \cdot 2 \Rightarrow B = \frac{1}{2}$
- $s = -1$：$1 = C \cdot 1 \cdot 1 \Rightarrow C = 1$
- $s = -2$：$1 = D \cdot 4 \cdot (-1) \Rightarrow D = -\frac{1}{4}$
- 比较 $s$ 系数：左边 $s$ 系数为 0，右边 $s$ 系数为 $2A + 3B$，故
  $$0 = 2A + 3B \Rightarrow 2A = -\frac{3}{2} \Rightarrow A = -\frac{3}{4}.$$

因此：
$$G(s) = -\frac{3}{4} \cdot \frac{1}{s} + \frac{1}{2} \cdot \frac{1}{s^2} + \frac{1}{s+1} - \frac{1}{4} \cdot \frac{1}{s+2}.$$

分别取逆变换：
$$\mathcal{L}^{-1}\left[\frac{1}{s}\right] = 1,$$
$$\mathcal{L}^{-1}\left[\frac{1}{s^2}\right] = t,$$
$$\mathcal{L}^{-1}\left[\frac{1}{s+1}\right] = e^{-t},$$
$$\mathcal{L}^{-1}\left[\frac{1}{s+2}\right] = e^{-2t}.$$

合并得：
$$g(t) = -\frac{3}{4} + \frac{1}{2}t + e^{-t} - \frac{1}{4}e^{-2t}.$$

**答案**：
$$\boxed{-\frac{3}{4} + \frac{1}{2}t + e^{-t} - \frac{1}{4}e^{-2t}}$$

## 三、变换综合填空题

本部分题目结合了傅里叶变换和拉普拉斯变换的应用，常见于考试后半部分。

---

### 题组：基本概念填空（2020级）

**题目**：说明物理现象初始状态的条件叫 ______；说明边界上约束情况的条件叫 ______. 如果没有给定解的条件，二阶线性齐次偏微分方程一般会有 ______ 个解；偏微分方程定解问题的适定性是指解的 ______.

**解答**：
- 说明物理现象初始状态的条件叫 **初始条件**（或初值条件）；
- 说明边界上约束情况的条件叫 **边界条件**（或边值条件）；
- 如果没有给定解的条件，二阶线性齐次偏微分方程一般会有 **无穷多** 个解；
- 偏微分方程定解问题的适定性是指解的 **存在性、唯一性、稳定性**.

**答案**：
$$\boxed{\text{初始条件（初值条件）；边界条件（边值条件）；无穷多；存在性、唯一性、稳定性}}$$

### 题2：极坐标下拉普拉斯算子（2020级）

**题目**：在极坐标系下，二维拉普拉斯方程表示为 ______.

**解答**：
通过极坐标变换 $x = r\cos\theta, y = r\sin\theta$，链式法则计算得：
$$\Delta u = \frac{\partial^2 u}{\partial r^2} + \frac{1}{r}\frac{\partial u}{\partial r} + \frac{1}{r^2}\frac{\partial^2 u}{\partial\theta^2}.$$

拉普拉斯方程 $\Delta u = 0$ 即：
$$\frac{\partial^2 u}{\partial r^2} + \frac{1}{r}\frac{\partial u}{\partial r} + \frac{1}{r^2}\frac{\partial^2 u}{\partial\theta^2} = 0.$$

也可写成：
$$\frac{1}{r}\frac{\partial}{\partial r}\left(r\frac{\partial u}{\partial r}\right) + \frac{1}{r^2}\frac{\partial^2 u}{\partial\theta^2} = 0.$$

**答案**：
$$\boxed{\frac{\partial^2 u}{\partial r^2} + \frac{1}{r}\frac{\partial u}{\partial r} + \frac{1}{r^2}\frac{\partial^2 u}{\partial\theta^2} = 0}$$

### 表3：重要定理速查

| 定理名称 | 内容 | 应用场景 |
|:---:|:---|:---|
| **卷积定理** | $\mathcal{F}[f*g] = \mathcal{F}[f]\cdot\mathcal{F}[g]$ | 乘积的逆变换 → 卷积 |

## 五、填空题解题技巧总结

### 傅里叶变换填空解题步骤

1. **确定变换约定**：本课程通常用 $\mathcal{F}[f](\alpha) = \int_{-\infty}^{+\infty} f(x)e^{-i\alpha x}dx$
2. **检查是否为分段函数**：是，则分段积分
3. **利用微分性质**：注意 $(i\alpha)^n$ 的符号，特别是 $n=2$ 时为 $-\alpha^2$
4. **回忆卷积定理**：乘积的逆变换 = 卷积

### 拉普拉斯变换填空解题步骤

1. **确定被变换函数类型**：是否含指数、三角函数、多项式乘积
2. **套用基本公式+性质**：位移性质、乘t性质是高频考点
3. **逆变换先分解**：部分分式分解是万能工具
4. **注意初值条件**：微分性质会引入初值

### 常考热点预测

| 考点 | 考查概率 | 典型考法 |
|:---:|:---:|:---|
| $\mathcal{L}[e^{-at}\cos bt]$ | ⭐⭐⭐⭐⭐ | 直接求变换 |
| $\mathcal{L}[te^{at}\sin\omega t]$ | ⭐⭐⭐⭐⭐ | 位移+乘t综合 |
| 卷积定理的逆用 | ⭐⭐⭐⭐ | $\mathcal{F}^{-1}[F(\alpha)G(\alpha)]$ |
| 部分分式分解逆变换 | ⭐⭐⭐⭐⭐ | $\mathcal{L}^{-1}[1/(s^2(s+a)(s+b))]$ |
| PDE的傅里叶变换 | ⭐⭐⭐⭐ | 热方程/拉普拉斯方程变换 |
