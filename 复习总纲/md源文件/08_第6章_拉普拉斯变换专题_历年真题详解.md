## `<a id="知识总结"></a>` 一、核心知识点总结

### 1. 拉普拉斯变换定义

**拉普拉斯变换**（像函数）：
$$\mathcal{L}[f(t)] = \bar{f}(s) = \int_{0}^{+\infty} f(t) e^{-st} dt$$

**适用范围**：定义在 $[0, +\infty)$ 上的函数（时间变量），指数增长型。

---

### 2. 拉普拉斯变换核心性质（必考！）

| 性质名称 | 公式 | 重要程度 | 应用场景 |
|:--------:|:----:|:--------:|:---------|
| **线性性质** | $\mathcal{L}[c_1 f + c_2 g] = c_1 \bar{f}(s) + c_2 \bar{g}(s)$ | ⭐⭐⭐ | 分解变换 |
| **位移性质** | $\mathcal{L}[e^{at} f(t)] = \bar{f}(s-a)$ | ⭐⭐⭐⭐⭐ | 指数乘函数 |
| **像函数微分性质** | $\mathcal{L}[t^n f(t)] = (-1)^n \dfrac{d^n}{ds^n}\bar{f}(s)$ | ⭐⭐⭐⭐ | 推特殊逆变换，如 $\dfrac{s}{(s^2+1)^2}$ |
| **微分性质** | $\mathcal{L}[f^{(n)}(t)] = s^n \bar{f}(s) - \sum_{k=0}^{n-1} s^{n-1-k} f^{(k)}(0)$ | ⭐⭐⭐⭐⭐ | **必考！含初值** |
| **积分性质** | $\mathcal{L}\!\left[\int_0^t f(\tau) d\tau\right] = \frac{\bar{f}(s)}{s}$ | ⭐⭐⭐⭐⭐ | **必考！积分方程** |
| **卷积定理** | $\mathcal{L}[(f*g)(t)] = \bar{f}(s) \bar{g}(s)$ | ⭐⭐⭐⭐⭐ | **核心！积分方程** |
| **延迟性质** | $\mathcal{L}[H(t-a) f(t-a)] = e^{-as} \bar{f}(s)$ | ⭐⭐⭐⭐ | 分段边界 |

> **卷积定义**：$(f * g)(t) = \int_{0}^{t} f(t-\tau) g(\tau) d\tau$

> **像函数微分性质常用特例**：若 $\mathcal{L}[f(t)] = \bar{f}(s)$，则
> $$\mathcal{L}[t f(t)] = -\frac{d}{ds}\bar{f}(s)$$
> 例如 $\mathcal{L}[\sin t]=\dfrac{1}{s^2+1}$，所以
> $$\mathcal{L}[t\sin t] = -\frac{d}{ds}\left(\frac{1}{s^2+1}\right)=\frac{2s}{(s^2+1)^2}$$
> 因此 $\mathcal{L}^{-1}\!\left[\dfrac{s}{(s^2+1)^2}\right]=\dfrac{t}{2}\sin t$。

---

### 3. 拉普拉斯逆变换方法

| 方法 | 适用场景 | 步骤 |
|:----:|:---------|:-----|
| **部分分式分解** | 有理函数 | 分解为最简分式 + 查表逆变换 |
| **Heaviside展开** | 一阶极点为主 | $\mathcal{L}^{-1}\!\left[\frac{F(s)}{G(s)}\right] = \sum \frac{F(\alpha_i)}{G'(\alpha_i)} e^{\alpha_i t}$ |
| **留数法** | 高阶极点、无穷多极点 | 计算各极点留数 $f(t) = \sum \text{Res}(e^{st} \bar{f}(s), s_k)$ |
| **卷积法** | 乘积形式像函数 | $\mathcal{L}^{-1}[F(s)G(s)] = \int_0^t f(t-\tau)g(\tau)d\tau$ |

## 题型一：常微分方程初值问题（E7, E8）

> **通用方法**：Laplace 变换 → 代数方程 → 部分分式分解 → 逆变换

---

### 🔵 【基础型】E8. 2013 第七题（10 分）

**题目**：求解 ODE 初值问题
$$y'' - 3y' + 2y = e^t \sin t, \quad y(0) = -3, \quad y'(0) = 5$$

📖 **知识点定位**：常系数二阶线性 ODE，非齐次项为指数乘三角函数

**解题思路**：
1. 求齐次通解（特征方程法）
2. 用待定系数法求特解
3. 应用初条件定常数

**Step 1：求齐次通解**

特征方程：$r^2 - 3r + 2 = 0 \Rightarrow (r-1)(r-2) = 0 \Rightarrow r_1 = 1, \, r_2 = 2$

$$y_h = C_1 e^t + C_2 e^{2t}$$

**Step 2：求特解**

非齐次项 $e^t\sin t$。由于 $1+i$ 不是特征根，设：
$$y_p = e^t(A\cos t + B\sin t)$$

代入方程比较系数得：$A = \frac{1}{2}, \, B = -\frac{1}{2}$

$$y_p = \frac{e^t}{2}(\cos t - \sin t)$$

**Step 3：应用初条件**

$$y = C_1 e^t + C_2 e^{2t} + \frac{e^t}{2}(\cos t - \sin t)$$

由 $y(0) = -3$：$C_1 + C_2 + \frac{1}{2} = -3 \Rightarrow C_1 + C_2 = -\frac{7}{2}$

由 $y'(0) = 5$：$C_1 + 2C_2 = 5$

解得：$C_1 = -12, \, C_2 = \frac{17}{2}$

$$\boxed{y(t) = -12 e^t + \frac{17}{2}\,e^{2t} + \frac{e^t}{2}(\cos t - \sin t)}$$

---

### 🟡 【参数讨论型】E7. 2014 第七题（12 分）

**题目**：求解 ODE 初值问题
$$y'' + \lambda y = f(t), \quad y(0) = a, \quad y'(0) = b$$
分 $\lambda = 0$，$\lambda = \omega^2 > 0$，$\lambda = -\mu^2 < 0$ 三种情况讨论。

📖 **知识点定位**：常系数二阶线性 ODE，参数讨论，卷积定理的 Duffing 原理

**解题思路**：
1. Laplace 变换得到代数方程
2. 分三种 $\lambda$ 情况讨论逆变换
3. 利用卷积定理写出非齐次特解的积分形式

**Step 1：Laplace 变换**

$$(s^2 + \lambda)Y - as - b = F(s) \Rightarrow Y = \frac{as + b}{s^2 + \lambda} + \frac{F(s)}{s^2 + \lambda}$$

**Step 2：分情况讨论**

| 情况 | 齐次解形式 | 卷积核 | 通解形式 |
|:----:|:---------:|:------:|:--------:|
| $\lambda = 0$ | $a + bt$ | $t$ | 齐次解 + $\int_0^t (t-\tau)f(\tau)d\tau$ |
| $\lambda = \omega^2 > 0$ | $a\cos\omega t + \frac{b}{\omega}\sin\omega t$ | $\frac{1}{\omega}\sin\omega t$ | 齐次解 + 卷积积分 |
| $\lambda = -\mu^2 < 0$ | $a\cosh\mu t + \frac{b}{\mu}\sinh\mu t$ | $\frac{1}{\mu}\sinh\mu t$ | 齐次解 + 卷积积分 |

**Step 3：写出各情况解**

✅ **$\lambda = 0$**：
$$\boxed{y(t) = a + bt + \int_0^t (t-\tau)\,f(\tau)\,d\tau}$$

✅ **$\lambda = \omega^2 > 0$**：
$$\boxed{y(t) = a\cos\omega t + \frac{b}{\omega}\sin\omega t + \frac{1}{\omega}\int_0^t \sin\omega(t-\tau)\,f(\tau)\,d\tau}$$

✅ **$\lambda = -\mu^2 < 0$**：
$$\boxed{y(t) = a\cosh\mu t + \frac{b}{\mu}\sinh\mu t + \frac{1}{\mu}\int_0^t \sinh\mu(t-\tau)\,f(\tau)\,d\tau}$$

## 题型二：积分方程（E3, E4）

> **通用方法**：识别卷积 → Laplace 变换 → 代数方程 → 部分分式 → 逆变换

---

### 🟡 【积分-微分型】E3. 2018 第五题（15 分）

**题目**：求解积分-微分方程
$$y'(t) = -2y(t) - \int_0^t y(s)\,ds, \quad y(0) = 1$$

📖 **知识点定位**：Volterra 积分-微分方程，Laplace 变换将积分转化为 $Y/s$

**解题思路**：
1. 识别积分项是卷积：$\int_0^t y(s)ds = 1 * y$
2. 微分性质自动包含初值
3. 解代数方程，部分分式，逆变换

**Step 1：Laplace 变换**

$$sY - 1 = -2Y - \frac{Y}{s}$$

**Step 2：解代数方程**

$$Y\!\left(s + 2 + \frac{1}{s}\right) = 1 \Rightarrow Y = \frac{s}{(s+1)^2}$$

**Step 3：部分分式分解**

$$Y = \frac{(s+1) - 1}{(s+1)^2} = \frac{1}{s+1} - \frac{1}{(s+1)^2}$$

**Step 4：逆变换**

$$y(t) = e^{-t} - t\,e^{-t}$$

$$\boxed{y(t) = (1-t)\,e^{-t}}$$

---

### 🟡 【纯积分型】E4. 2017 第四题（12 分）

**题目**：求解 Volterra 积分方程
$$y(t) = \cos t - \int_0^t (t-\tau)\,y(\tau)\,d\tau$$

📖 **知识点定位**：第二类 Volterra 积分方程，核 $(t-\tau)$ 是卷积核

**解题思路**：
1. 识别积分是 $t * y$ 卷积
2. Laplace 变换后解代数方程
3. 部分分式分解 + 特殊像函数逆变换

**Step 1：识别卷积并变换**

$$y = \cos t - (t * y) \Rightarrow Y = \frac{s}{s^2+1} - \frac{1}{s^2}Y$$

**Step 2：解代数方程**

$$Y\!\left(1 + \frac{1}{s^2}\right) = \frac{s}{s^2+1} \Rightarrow Y = \frac{s^3}{(s^2+1)^2}$$

**Step 3：部分分式分解**

$$Y = \frac{s(s^2+1) - s}{(s^2+1)^2} = \frac{s}{s^2+1} - \frac{s}{(s^2+1)^2}$$

**Step 4：逆变换**

利用特殊公式：$\mathcal{L}^{-1}\!\left[\dfrac{s}{(s^2+1)^2}\right] = \dfrac{t}{2}\sin t$

$$y(t) = \cos t - \frac{t}{2}\sin t$$

$$\boxed{y(t) = \cos t - \frac{t}{2}\,\sin t}$$

---

## 题型三：一阶线性 PDE（E1, E2）

> **通用方法**：对 $t$ 作 Laplace 变换 → 关于 $x$ 的一阶线性 ODE → 积分因子法 → 部分分式 → 逆变换

---

### 🟡 【PDE 降阶型】E1. 2020 第七题（10 分）

**题目**：求解 PDE 初边值问题
$$x\,u_x + u_t = x^2 t, \quad u(x,0) = x^2, \quad u(0,t) = 0$$

📖 **知识点定位**：一阶线性 PDE，Laplace 变换降维为 ODE

**解题思路**：
1. 对 $t$ 变换，$u_t \to sU - x^2$
2. 得到关于 $x$ 的一阶线性 ODE
3. 积分因子法求解
4. 部分分式分解 + 逆变换

**Step 1：Laplace 变换**

$$x\,U_x + sU - x^2 = \frac{x^2}{s^2} \Rightarrow x\,U_x + sU = x^2\!\left(1 + \frac{1}{s^2}\right)$$

**Step 2：积分因子法**

积分因子 $\mu = x^s$。解得：

$$U(x,s) = \frac{x^2}{s+2}\!\left(1 + \frac{1}{s^2}\right)$$

**Step 3：部分分式分解**

$$U = x^2\!\left(-\frac{1}{4s} + \frac{1}{2s^2} + \frac{5}{4(s+2)}\right)$$

**Step 4：逆变换**

$$\boxed{u(x,t) = x^2\!\left(\frac{t}{2} - \frac{1}{4} + \frac{5}{4}\,e^{-2t}\right)}$$

---

### 🟡 【PDE 变体】E2. 2019 第七题（12 分）

**题目**：求解 PDE 初边值问题
$$x\,u_x + u_t = x^2 t^2, \quad u(x,0) = x^2, \quad u(0,t) = 0$$

📖 **知识点定位**：与 E1 同构，仅源项由 $x^2 t$ 变为 $x^2 t^2$

**Step 1：Laplace 变换**

$\mathcal{L}[t^2] = 2/s^3$

$$x\,U_x + sU = x^2\!\left(1 + \frac{2}{s^3}\right)$$

**Step 2：积分因子求解**

$$U(x,s) = \frac{x^2}{s+2}\!\left(1 + \frac{2}{s^3}\right)$$

**Step 3：部分分式分解**

$$U = x^2\!\left(\frac{1}{4s} - \frac{1}{2s^2} + \frac{1}{s^3} + \frac{3}{4(s+2)}\right)$$

**Step 4：逆变换**

$$\boxed{u(x,t) = x^2\!\left(\frac{t^2}{2} - \frac{t}{2} + \frac{1}{4} + \frac{3}{4}\,e^{-2t}\right)}$$

## 题型四：半无界波动方程（E5, E6）

> **通用方法**：对 $t$ 作 Laplace 变换 → 二阶常系数 ODE → 特解+齐次解 → 有界性+边界定常数 → 延迟性质逆变换

---

### 🟠 【波动+有界性】E5. 2016 第五题（12 分）

**题目**：求解半无界波动方程
$$u_{tt} = a^2(u_{xx} + \cos\omega t), \quad x > 0,\; t > 0$$
$$u(0,t) = 0, \quad u(x,0) = 0, \quad u_t(x,0) = 0, \quad u \text{ 有界}$$

📖 **知识点定位**：半无界弦受迫振动，Laplace 变换 + 有界性条件 + 延迟性质

**解题思路**：
1. 对 $t$ 变换，得到关于 $x$ 的二阶 ODE
2. 求常数特解 + 齐次解
3. 有界性舍去 $e^{sx/a}$ 项
4. 边界 $u(0,t)=0$ 定常数
5. 部分分式 + 延迟性质逆变换

**Step 1：Laplace 变换**

$$s^2 U = a^2 U_{xx} + \frac{a^2 s}{s^2+\omega^2} \Rightarrow U_{xx} - \frac{s^2}{a^2}\,U = -\frac{s}{s^2+\omega^2}$$

**Step 2：特解**

$$U_p = \frac{a^2}{s(s^2+\omega^2)}$$

**Step 3：通解与边界条件**

$$U = C_1 e^{-sx/a} + C_2 e^{sx/a} + U_p$$

有界性 $\Rightarrow C_2 = 0$。$U(0,s)=0 \Rightarrow C_1 = -U_p$

$$U(x,s) = \frac{a^2}{s(s^2+\omega^2)}\!\left(1 - e^{-sx/a}\right)$$

**Step 4：逆变换 + 延迟性质**

$$\mathcal{L}^{-1}\!\left[\frac{a^2}{s(s^2+\omega^2)}\right] = \frac{a^2}{\omega^2}(1 - \cos\omega t)$$

$$u(x,t) = \begin{cases} \dfrac{a^2}{\omega^2}(1 - \cos\omega t), & t < \dfrac{x}{a} \\[8pt] \dfrac{a^2}{\omega^2}\!\left[\cos\omega\!\left(t - \dfrac{x}{a}\right) - \cos\omega t\right], & t \geq \dfrac{x}{a} \end{cases}$$

---

### 🟠 【波动+有界性】E6. 2015 第五题（12 分）

**题目**：求解半无界波动方程
$$u_{tt} = a^2(u_{xx} + \sin\omega t), \quad x > 0,\; t > 0$$
$$u(0,t) = 0, \quad u(x,0) = 0, \quad u_t(x,0) = 0, \quad u \text{ 有界}$$

📖 **知识点定位**：与 E5 同构，源项由 $\cos\omega t$ 改为 $\sin\omega t$

**Step 1：Laplace 变换**

$$U(x,s) = \frac{a^2\omega}{s^2(s^2+\omega^2)}\!\left(1 - e^{-sx/a}\right)$$

**Step 2：逆变换**

$$\mathcal{L}^{-1}\!\left[\frac{a^2\omega}{s^2(s^2+\omega^2)}\right] = \frac{a^2}{\omega^2}(\omega t - \sin\omega t)$$

$$u(x,t) = \begin{cases} \dfrac{a^2}{\omega^2}(\omega t - \sin\omega t), & t < \dfrac{x}{a} \\[8pt] \dfrac{a^2}{\omega^2}\!\left[-\omega\cdot\frac{x}{a} - \sin\omega\!\left(t - \dfrac{x}{a}\right) + \sin\omega t\right], & t \geq \dfrac{x}{a} \end{cases}$$

---

### 🟠 **【基础经典型】E9. 2023 第五题（12分）**

**题目：** 用 Laplace 变换法求解波动方程半无界定解问题
$$
\begin{cases}
u_{tt}=a^2u_{xx}, & x>0,\ t>0,\\
u(x,0)=0,\quad u_t(x,0)=0, & x\ge0,\\
u(0,t)=f(t),\quad \lim\limits_{x\to+\infty}u(x,t)=0, & t\ge0.
\end{cases}
$$

📖 **知识点定位：** 半无界齐次波动方程，边界扰动传播，延迟性质的典型应用。

**解题思路：**
1. 对时间 $t$ 做 Laplace 变换，得到关于 $x$ 的常微分方程
2. 求解常微分方程，利用有界性舍去增长解
3. 利用边界条件确定常数
4. 利用延迟性质做 Laplace 反变换

---

**Step 1：对时间 $t$ 做 Laplace 变换**

记
$$
U(x,s)=\mathcal L_t[u(x,t)],\qquad F(s)=\mathcal L[f(t)].
$$

由于初值为零，
$$
\mathcal L[u_{tt}]=s^2U.
$$

因此方程变为
$$
s^2U=a^2U_{xx}.
$$

即
$$
U_{xx}-\frac{s^2}{a^2}U=0.
$$

---

**Step 2：解关于 $x$ 的常微分方程**

方程
$$
U_{xx}-\frac{s^2}{a^2}U=0
$$

的通解为
$$
U(x,s)=C_1(s)e^{sx/a}+C_2(s)e^{-sx/a}.
$$

由于 $x\to+\infty$ 时要求解有界并趋于 0，且 $\operatorname{Re}s>0$，必须令
$$
C_1(s)=0.
$$

所以
$$
U(x,s)=C_2(s)e^{-sx/a}.
$$

---

**Step 3：利用边界条件确定常数**

由
$$
u(0,t)=f(t),
$$

得
$$
U(0,s)=F(s).
$$

而
$$
U(0,s)=C_2(s),
$$

所以
$$
C_2(s)=F(s).
$$

因此
$$
U(x,s)=F(s)e^{-sx/a}.
$$

---

**Step 4：Laplace 反变换**

利用延迟性质：
$$
\mathcal L^{-1}\left[e^{-cs}F(s)\right]=H(t-c)f(t-c),
$$

其中 $H$ 是 Heaviside 函数。

这里
$$
c=\frac{x}{a}.
$$

所以
$$
\boxed{
u(x,t)=H\left(t-\frac{x}{a}\right)f\left(t-\frac{x}{a}\right).}
$$

也可以写成分段形式：
$$
\boxed{
 u(x,t)=
\begin{cases}
0, & 0\le t<\dfrac{x}{a},\\[6pt]
f\left(t-\dfrac{x}{a}\right), & t\ge\dfrac{x}{a}.
\end{cases}}
$$

---

**💡 物理意义：**
边界 $x=0$ 的扰动 $f(t)$ 以速度 $a$ 向右传播。位置 $x$ 处要等到
$$
t=\frac{x}{a}
$$
以后才能感受到边界扰动，因此解中出现延迟量 $t-\frac{x}{a}$。

## `<a id="变体"></a>` 四、题型变体规律总结

### 拉普拉斯变换四大题型谱系

```
Laplace 变换法
│
├── 常微分方程初值问题
│   ├── E8 (2013)：二阶线性 ODE + 非齐次 e^t sin t
│   └── E7 (2014)：二阶 ODE + 三参数讨论
│
├── 积分方程
│   ├── E3 (2018)：积分-微分方程 y' = -2y - ∫y
│   └── E4 (2017)：纯积分方程 y = cos t - (t*y)
│
├── 一阶线性 PDE（降阶法）
│   ├── E1 (2020)：xu_x + u_t = x² t
│   └── E2 (2019)：xu_x + u_t = x² t²（同构变体）
│
└── 半无界波动方程（延迟性质）
    ├── E5 (2016)：u_tt = a²(u_xx + cosωt)
    ├── E6 (2015)：u_tt = a²(u_xx + sinωt)（同构变体）
    └── E9 (2023)：u_tt = a²u_xx，齐次方程 + 边界扰动
```

## `<a id="模板"></a>` 五、解题通用模板

### 模板 A：常微分方程初值问题

**题目形式**：$y'' + ay' + by = f(t), \quad y(0)=c_1, \, y'(0)=c_2$

```
第1步：Laplace 变换
   s²Y - sy(0) - y'(0) + a(sY - y(0)) + bY = F(s)
   
第2步：解出 Y(s)
   Y(s) = 有理函数 = P(s) / Q(s)
   
第3步：部分分式分解
   - 一阶实根：A/(s-r)
   - 二阶共轭复根：(As+B)/(s²+ps+q)
   - 重根：A/(s-r)^k
   
第4步：逐项逆变换查表
   1/s ← 1, 1/s² ← t, 1/(s-a) ← e^(at), ...
```

---

### 模板 B：积分-微分方程

**题目形式**：$y' + ay + \int_0^t k(t-\tau)y(\tau)d\tau = f(t)$

```
第1步：识别卷积项
   ∫k(t-τ)y(τ)dτ = k * y
   
第2步：Laplace 变换
   sY - y(0) + aY + K(s)Y = F(s)
   
第3步：解代数方程求 Y(s)
   Y(s) = (y(0) + F(s)) / (s + a + K(s))
   
第4步：部分分式分解 + 逆变换
```

---

### 模板 C：一阶线性 PDE（降阶法）

**题目形式**：$a(x)u_x + u_t = f(x,t)$，带初边值

```
第1步：对 t 作 Laplace 变换
   a(x)U_x + sU = u(x,0) + F(x,s)
   
第2步：解关于 x 的一阶线性 ODE
   标准形式：U_x + P(x)U = Q(x)
   积分因子：μ(x) = e^(∫P(x)dx)
   
第3步：应用边界条件定常数
   如 u(0,t)=0 ⇒ U(0,s)=0
   
第4步：部分分式分解 + 逐项逆变换
```

---

### 模板 D：半无界波动方程

#### 题型一：有源型（齐次边界）
**题目形式**：$u_{tt} = a^2u_{xx} + f(t)$，$u(0,t)=0$，初值为 0，有界  
→ 对应 E5 (2016)、E6 (2015)

```
第1步：对 t 作 Laplace 变换
   s²U = a²U_xx + F(s)
   
第2步：求常数特解 U_p
   令 U_p = 常数，代入得 U_p = F(s)a²/s²
   
第3步：通解
   U = C1 e^(-sx/a) + C2 e^(sx/a) + U_p
   
第4步：定常数
   有界性 ⇒ C2 = 0
   边界 U(0,s)=0 ⇒ C1 = -U_p
   
第5步：写成 U = U_p(1 - e^(-sx/a))
   对 U_p 做部分分式后逆变换
   对 U_p e^(-sx/a) 用延迟性质
```

---

#### 题型二：齐次方程边界扰动型
**题目形式**：$u_{tt} = a^2u_{xx}$，$u(0,t)=f(t)$，初值为 0，有界  
→ 对应 E9 (2023)

```
第1步：对 t 作 Laplace 变换
   s²U = a²U_xx
   
第2步：通解
   U = C1 e^(-sx/a) + C2 e^(sx/a)
   
第3步：定常数
   有界性 ⇒ C2 = 0
   边界 U(0,s)=F(s) ⇒ C1 = F(s)
   
第4步：写成 U = F(s) e^(-sx/a)
   直接用延迟性质逆变换
```

## ⚠️ 常见易错点提醒

1. **微分性质别忘初值！** $\mathcal{L}[y'] = sY - y(0)$，不是 $sY$

2. **积分方程的卷积识别**：$\int_0^t k(t-\tau)y(\tau)d\tau = K(s)Y(s)$，不是 $K(s)/Y(s)$

3. **延迟性质别丢阶跃函数**：$\mathcal{L}^{-1}[e^{-as}F(s)] = f(t-a)H(t-a)$，注意 $f$ 的自变量是 $t-a$

4. **有界性条件定常数**：半无界问题中 $e^{sx/a}$ 当 $x\to\infty$ 无界，舍去 $C_2$

5. **部分分式分解系数要验算**：通分后比较分子，别算错系数
